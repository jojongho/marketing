# 🔖 레퍼런스

외부 참조 자료, 경쟁사 분석, 업계 트렌드, 기술 문서

## 📁 하위 폴더

### 경쟁사_분석
- 경쟁 분양 현장 웹사이트
- 광고 크리에이티브
- 마케팅 전략 분석
- 차별화 포인트 도출

### 업계_트렌드
- 부동산 시장 동향
- 분양 마케팅 트렌드
- 디지털 마케팅 변화
- 소비자 행동 패턴

### 마케팅_사례
- 성공 사례 연구
- 캠페인 벤치마킹
- 크리에이티브 아이디어
- 전략 참고

### 기술_문서
- Figma 튜토리얼
- GA4 가이드
- 광고 플랫폼 매뉴얼
- 개발 참고 자료

## 📝 작성 가이드

### 템플릿 사용
`/templates/레퍼런스_템플릿.md` 복사

### 파일명 규칙
```
[카테고리]_제목_출처.md

예시:
경쟁사_XX아파트_랜딩페이지_분석.md
사례_네이버_검색광고_성공사례.md
트렌드_2024_부동산_마케팅_동향.md
```

### 필수 포함 사항
- **출처**: URL, 저자, 날짜
- **요약**: 핵심 내용 정리
- **적용 가능성**: 우리 프로젝트에 어떻게?
- **액션 아이템**: 구체적 실행 항목

## 🎯 수집 기준

### 우선순위 높음
- 직접 적용 가능한 전략/기법
- 천안아산 지역 관련 정보
- 검증된 성과 데이터
- 최신 트렌드 (6개월 이내)

### 선택적 수집
- 일반적인 마케팅 이론
- 타 산업 사례 (응용 필요)
- 과거 자료 (참고용)

## 🔍 효과적인 활용법

### 1. 경쟁사 분석
**목적**: 차별화 포인트 발견

**체크리스트**:
- [ ] 랜딩페이지 구조
- [ ] 헤드라인 및 카피
- [ ] 비주얼 스타일
- [ ] CTA 전략
- [ ] 광고 키워드
- [ ] SNS 활동

**결과물**:
- 우리의 차별화 전략
- 피해야 할 함정
- 벤치마킹 요소

### 2. 사례 연구
**목적**: 검증된 방법론 학습

**체크리스트**:
- [ ] 성과 데이터 확인
- [ ] 성공 요인 분석
- [ ] 적용 가능성 평가
- [ ] 리스크 검토

**결과물**:
- 적용 계획
- 수정/보완 사항
- 테스트 방법

### 3. 트렌드 파악
**목적**: 시장 변화 대응

**모니터링 영역**:
- 부동산 시장 동향
- 금리/정책 변화
- 소비자 선호 변화
- 마케팅 기술 발전

**결과물**:
- 기회/위협 분석
- 전략 조정
- 새로운 시도
