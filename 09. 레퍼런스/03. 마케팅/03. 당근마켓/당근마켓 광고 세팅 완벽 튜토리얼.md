## 1단계: 당근 비즈니스 계정 생성
### 🔗 접속 경로
1. 구글에서 **"당근 비즈니스"** 검색
2. 또는 직접 접속: https://business.daangn.com

### 📝 회원가입 절차
#### Step 1. 시작하기 클릭
```
당근 비즈니스 홈페이지 → [시작하기] 버튼
```

#### Step 2. 로그인/회원가입
- 기존 당근 계정이 있다면: 로그인
- 없다면: 새 계정 생성 (휴대폰 인증)

#### Step 3. 비즈니스 프로필 등록
```
필수 입력 항목:
- 사업자명 (상호명)
- 사업자등록번호
- 업종 선택
- 사업장 주소
- 대표 전화번호
- 대표 이미지 (로고 또는 대표 사진)
```

**⚠️ 주의사항**:
- 사업자등록증과 정보 일치 필수
- 대표 이미지는 **직접 촬영한 사진** 권장 (클릭률 3배 차이)
- 주소는 네이버 플레이스와 동일하게 입력

## 2단계: 광고 계정 만들기
### 🏗️ 광고 계정 생성
#### Step 1. 전문가 모드 진입
```
당근 비즈니스 로그인 → 상단 [광고] 메뉴 → [새 광고 계정 만들기]
```

#### Step 2. 광고 계정 정보 입력
```
광고 계정명: 내 업체명 (예: "서울 강남 부동산")
```

#### Step 3. 결제 수단 등록
```
[결제 수단 추가] → 신용카드 또는 자동이체 선택
→ 카드 정보 입력 → 등록 완료
```

**💰 캐시 충전**:
- 첫 광고 시작 전 최소 **5만원** 충전 권장
- 하루 예산 x 7일 = 1주일 테스트 가능 금액

## 3단계: 피드 광고 세팅
> **피드 광고**: 당근 홈 화면에 노출되는 광고 (잠재 고객 발견)
### 🎯 캠페인 만들기
#### Step 1. 광고 유형 선택
```
[캠페인 만들기] → [피드 광고] 선택
```

#### Step 2. 캠페인 목적 선택
```
✅ 오프라인 사업장: [웹사이트 방문 늘리기]
→ [연동하지 않음] (추적 코드 불필요)
```

**⚠️ 선택 가이드**:
- **리드폼 광고**: DB 수집 (보험, 대출, 렌트카) - 부동산은 부적합
- **웹사이트 전환**: 온라인 쇼핑몰 전용
- **웹사이트 방문**: 오프라인 → 네이버 플레이스 유도


---

### 🎯 광고 그룹(세트) 설정
#### Step 1. 타겟 설정 - 지역
```
📍 지역 선택 방법:

1. [특정 지역에 게재하기] 선택
2. 시/구/동 검색
   예: "서울시 강남구 역삼동" 입력
3. [추가] 클릭

💡 지역 범위 전략:
- 도보 고객 (카페, 식당): 반경 1km (인접 2~3개 동)
- 차량 고객 (병원, 학원): 반경 3km (전체 구)
- 부동산: 타겟 아파트 단지 중심 반경 5km
```

**⚠️ GPS 주의사항**:
- GPS는 24시간 실시간 업데이트 **안 됨**
- 너무 타이트하게 잡으면 노출 안 됨
- 처음엔 넓게 → 성과 보고 좁히기

#### Step 2. 타겟 설정 - 성별/연령

```
성별:
- 부동산: 모든 성별
- 여성 타겟 (미용실, 네일샵): 여성
- 남성 타겟 (이발소): 남성

연령:
- 전체 연령 또는 핵심 고객층 (예: 30~50대)
```

#### Step 3. 타겟 설정 - 관심사 (선택)
```
💡 관심사 전략:

✅ 관심사 넣는 경우:
- 명확한 니치 시장 (예: 골프 레슨 → 골프 관심사)
- 특정 라이프스타일 (예: 비건 레스토랑 → 건강/웰빙)

❌ 관심사 안 넣는 경우:
- 지역 기반 필수 서비스 (부동산, 동네 식당, 편의점)
- 이미 관심 있는 사람보다 신규 고객 발굴이 목표

⚠️ 함정 주의:
헬스장 광고에 "보디빌딩" 관심사 넣으면?
→ 이미 운동하는 헬창만 타게팅 (신규 고객 놓침)
```

**추천 전략**:
1. 관심사 넣은 그룹 1개
2. 관심사 안 넣은 그룹 1개
3. 1주일 후 성과 비교 → 좋은 쪽 남기기

#### Step 4. 예산 설정
```
💰 예산 설정 방법:

방법 1: 일일 예산 (권장)
- 하루 예산: 10,000원
- 종료일: 없음 (계속 광고하기 체크)

방법 2: 총 예산 (소액 광고 꿀팁)
- 하루 5,000원 쓰고 싶은데 설정 안 됨?
- 총 예산: 50,000원
- 기간: 10일 (7월 10일 ~ 7월 20일)
- 결과: 하루 5,000원씩 10일간 집행!
```

**💡 예산 가이드**:
- 최소 **일 5,000원** (테스트용)
- 권장 **일 10,000원** (안정적 노출)
- 적극 **일 30,000원** (빠른 데이터 수집)

#### Step 5. 입찰 방식
```
처음 시작: [자동 입찰] 선택
1주일 후: [수동 입찰]로 전환 (비용 절감)

💡 수동 입찰 전환 시점:
- 클릭당 평균 비용 파악 후
- 경쟁사 입찰가 패턴 분석 후
- 노출량 충분히 확보된 후
```

#### Step 6. 광고 일정
```
✅ 첫 1주일: [모든 시간 광고]
→ 데이터 수집 위해 전체 시간 노출

📊 1주일 후 최적화:
- 월~목: 오전 9시 ~ 오후 6시 (저강도)
- 금~일: 전체 시간 (고강도)
- 피크 시간대만 집중 (예: 저녁 7~10시)
```

### 🎨 광고 소재 만들기

#### Step 1. 홍보 대상 선택

```
선택지:
1. 비즈 프로필 홈 (권장)
   → 지속적인 브랜딩, 전체 서비스 소개

2. 소식
   → 특정 이벤트/프로모션 홍보
   → 사전에 소식 발행 필요
```

**⚠️ 첫 광고는 "비즈 프로필 홈" 추천**
#### Step 2. 대표 사진 선택
```
✅ 클릭률 높은 사진 조건:
1. 직접 촬영한 사진 (클릭률 3배 차이)
2. 1:1 비율 (정사각형)
3. 글자 최소화 (가독성 확보)
4. 서비스/상품 생생하게 담기

❌ 피해야 할 사진:
- 캔바/망고보드 템플릿 (광고티 심함)
- 글자 가득한 전단지 스타일
- 저화질 이미지
- 타업체 로고/워터마크
```

**부동산 예시**:
- ✅ 매물 단지 전경, 깔끔한 인테리어, 동네 전망
- ❌ 빨간 글씨로 "특급 매물!!!" 도배된 전단지

#### Step 3. 광고 제목 작성
```
📝 클릭률 높이는 제목 공식:

공식 1: 궁금증 유발
- "소개팅 요즘 여기서 한다던데?"
- "우리 부모님 여기 모시고 가니까 되게 좋아하시던데?"

공식 2: 혜택 강조 (분할 가격)
- ❌ "보컬레슨 월 25만 원"
- ✅ "하루 8,000원으로 취미 생활 시작하세요"

공식 3: 타게팅 명확화
- "신혼부부 전세 알아보시나요?"
- "역세권 투자 고민 중이신가요?"

💡 제목 작성 원칙:
- 혜택은 많게, 비용은 적게
- 물음표(?) 활용으로 대화 느낌
- 20자 이내 (모바일 화면 고려)
```

#### Step 4. 브랜드 이름 & CTA

```
브랜드 이름: 상호명 그대로 입력
→ 사업자 명칭과 일치해야 함

CTA 버튼:
- ✅ "더 알아보기" (추천)
- ✅ "문의하기"
- ❌ "구매하기" (광고티)
- ❌ "주문하기" (강압적)
```

#### Step 5. 웹사이트 URL (중요!)
```
🚨 네이버 플레이스 원 주소 입력 방법:

❌ 잘못된 방법:
주소창 URL 복사 → 파라미터 포함 → 부정 트래픽 오해

✅ 올바른 방법:
1. 네이버 플레이스 페이지 접속
2. [공유] 버튼 클릭
3. 공유 URL 복사 (예: https://naver.me/xxxxx)
4. 당근 광고에 붙여넣기
5. 자동으로 원 주소로 변환됨
   (https://m.place.naver.com/place/숫자)
6. ❗ 물음표(?) 이전까지만 사용

예시:
원 주소: https://m.place.naver.com/place/1234567890
잘못된 주소: ...place/1234567890?searchType=...
```

**⚠️ 이거 안 지키면**: 네이버 플레이스 순위 하락 가능 (부정 트래픽 의심)

#### Step 6. 소재 복사 & 다중 소재
```
💡 전문가 모드의 핵심: A/B 테스트

1개 그룹에 소재 3~5개 넣기:
- 소재 A: 인테리어 사진 + 제목1
- 소재 B: 외관 사진 + 제목2
- 소재 C: 서비스 이미지 + 제목3

→ 당근이 자동으로 성과 좋은 소재에 예산 집중!

📋 체크리스트:
- [ ] 같은 내용, 다른 이미지 3개
- [ ] 같은 이미지, 다른 제목 3개
- [ ] 전혀 다른 컨셉 3개
```

## 4단계: 검색 광고 세팅

> **검색 광고**: 사용자가 키워드 검색할 때 상단 노출 (구매 의도 높음)

### 🔍 검색 광고 차별점
- **피드 광고**: "우리 가게 있어요" (인지도)
- **검색 광고**: "네일샵 찾는데요?" → "여기요!" (전환)

---

### 🎯 캠페인/그룹 설정

```
Step 1~4는 피드 광고와 동일
- 웹사이트 방문 늘리기
- 지역 타겟팅
- 성별/연령
- 예산 설정
```

---

### 🔑 검색 키워드 등록 (핵심!)

#### Step 1. 키워드 찾기

```
💡 키워드 발굴 방법:

1. 직접 입력:
   - 당근 앱에서 내 업종 검색
   - 자동완성 키워드 확인
   - 경쟁사가 쓰는 키워드 파악

2. 당근 키워드 제안:
   - 검색 키워드 입력 시 추천 키워드 표시
   - "검색량 충분" 표시 확인

3. 네이버 키워드 도구 활용:
   - 네이버 광고 키워드 도구
   - 당근에도 비슷한 검색 패턴
```

**부동산 예시**:
```
핵심 키워드:
- 역삼동 부동산
- 강남 전세
- 테헤란로 원룸
- 역삼역 부동산
- 선릉역 오피스텔

롱테일 키워드:
- 역삼동 신축 오피스텔
- 강남 전세 2억
- 테헤란로 역세권
```

#### Step 2. 키워드 등록 전략

```
✅ 키워드 선택 기준:

1. 검색량 기준:
   - "검색량 충분히 높음" ✅
   - "검색량 적음" → ⚠️ 롱테일은 포함

2. 키워드 개수:
   - 최대 100개 등록 가능
   - 권장: 20~50개 (핵심 + 롱테일 믹스)

3. 키워드 분류:
   - 지역명 (역삼동, 강남, 테헤란로)
   - 서비스명 (부동산, 공인중개사, 중개사무소)
   - 매물 유형 (전세, 월세, 원룸, 투룸, 오피스텔)
   - 조합 (역삼동 전세, 강남 원룸)
```

**💡 키워드 조합 예시** (부동산):
```
지역 x 서비스:
- 역삼동 부동산
- 강남구 공인중개사
- 테헤란로 중개사무소

지역 x 매물:
- 역삼동 전세
- 강남 원룸
- 선릉역 오피스텔

지역 x 매물 x 조건:
- 역삼동 전세 1억
- 강남 원룸 월세
- 테헤란로 신축 오피스텔
```

---

### 💰 클릭당 입찰가 설정

#### 입찰가 원리 이해

```
🎯 경쟁 입찰 방식:

예시:
1등: 200원 입찰
2등: 150원 입찰
3등: 100원 입찰

실제 청구 금액:
1등 → 160원 (2등 + 10원)
2등 → 110원 (3등 + 10원)

💡 핵심:
- 1등 하려고 500원 넣어도 실제 160원만 청구
- BUT 자동 입찰은 더 비싸게 책정함
```

#### 입찰가 전략

```
초기 (1주일):
- 당근 추천 [평균 단가] 그대로 사용
- 데이터 수집 우선

최적화 (1주일 후):
- 평균 단가의 50~70% 수동 입찰
- 검색량 많은 키워드 → 낮은 입찰가 (경쟁 치열)
- 검색량 적은 롱테일 → 높은 입찰가 (노출 확보)

예시:
- "강남 부동산" (검색량 높음) → 150원
- "역삼동 신축 오피스텔" (롱테일) → 300원
```

### 🎨 검색 광고 소재

```
피드 광고와 거의 동일하나 차이점:

✅ 제목 차별화:
- 피드: "우리 부모님 여기 모시고 가니까 좋아하시던데?"
- 검색: "역삼동 전세 찾으세요? 실거래가 기반 매물 추천"

✅ 소개 문구 (추가):
- 검색 의도에 맞는 구체적 혜택
- "첫 방문 고객 중개수수료 10% 할인"
- "전세 대출 상담 무료"
```

## 5단계: 광고 소재 최적화

### 📸 이미지 소재 가이드

#### 이미지 사양
```
필수:
- 비율: 1:1 (정사각형)
- 최소 해상도: 800 x 800px
- 권장 해상도: 1200 x 1200px
- 파일 형식: JPG, PNG
- 용량: 5MB 이하

동영상:
- 비율: 1:1 또는 9:16 (세로 영상)
- 길이: 6~60초
- 용량: 100MB 이하
```

#### 클릭률 높은 이미지 조건

```
✅ 이렇게 하세요:
1. 직접 촬영 (스톡 이미지 X)
2. 밝고 깔끔한 색감
3. 주인공 뚜렷 (서비스/상품 명확)
4. 글자 최소화 (있어도 1~2줄)
5. 여백 충분 (답답하지 않게)

❌ 이건 피하세요:
1. 캔바/망고보드 템플릿 (광고티)
2. 빨간색 "특가!!!" 도배
3. 복잡한 배경
4. 너무 많은 텍스트
5. 저화질/흐릿한 이미지
```

**부동산 이미지 전략**:
```
A형: 매물 사진
- 아파트 외관, 인테리어, 전망
- 실제 거주자 입장에서 매력적인 앵글

B형: 지역 사진
- 역세권 표시, 주변 인프라
- "역삼역 도보 3분" 같은 입지 강조

C형: 서비스 사진
- 친절한 상담 장면
- 사무실 내부 (신뢰감)

💡 3가지 다 테스트 → 클릭률 높은 거 채택
```

---

### 📝 카피라이팅 공식

#### 제목 템플릿

```
템플릿 1: 궁금증 유발형
"[타겟]이 요즘 [행동] 한다던데?"
예: "신혼부부들이 요즘 여기서 전세 구한다던데?"

템플릿 2: 혜택 강조형
"[혜택] + [비용 분할]"
예: "역삼역 도보 3분, 하루 3만 원이면 전세 보증금 걱정 끝"

템플릿 3: 타게팅 직격형
"[타겟] [상황]이시죠?"
예: "강남 전세 알아보시나요?"

템플릿 4: 해결책 제시형
"[문제] 때문에 고민이시라면"
예: "높은 중개수수료 때문에 고민이시라면"
```

#### 소개 문구 (검색 광고)

```
공식: 특징 + 혜택 + CTA

예시 (부동산):
"역삼동 10년 경력 공인중개사의
실거래가 기반 정직한 매물 추천.
첫 방문 고객 중개수수료 10% 할인!"

구조:
1. 차별점 (10년 경력, 실거래가 기반)
2. 신뢰 요소 (공인중개사, 정직한)
3. 혜택 (중개수수료 할인)
```

## 6단계: 광고 승인 & 모니터링

### ✅ 광고 검수

```
소요 시간: 약 1~2시간 (영업일 기준)

승인 기준:
- 사업자 정보 일치
- 과장/허위 광고 없음
- 비방/차별적 표현 없음
- 성인/도박 콘텐츠 없음

❌ 반려 사유:
- 타업체 로고/상표 무단 사용
- 과도한 할인 표현 ("100% 할인")
- 의료법 위반 (병원 광고)
- 오타/맞춤법 오류
```

---

### 📊 성과 체크 (첫 1주일)

```
일일 체크리스트:

1. 예산 소진 속도
   - 너무 빠름: 입찰가 낮추기
   - 너무 느림: 타겟 범위 확대

2. 클릭률 (CTR)
   - 1% 이상: 양호
   - 0.5% 이하: 소재 교체

3. 클릭당 비용 (CPC)
   - 당근 평균: 200~500원
   - 업종별 차이 있음

4. 네이버 플레이스 유입
   - 당근 유입 통계 확인
   - 외부 유입자 점수 상승 여부
```

## 트러블슈팅

### ❌ 광고가 노출 안 돼요

```
원인 1: 예산 소진
→ 캐시 충전 확인

원인 2: 지역 범위 너무 좁음
→ 반경 넓히기 (1km → 3km)

원인 3: 입찰가 너무 낮음
→ 자동 입찰로 변경

원인 4: 광고 검수 중
→ 1~2시간 대기
```

---

### ❌ 클릭은 많은데 문의가 없어요

```
원인 1: 타게팅 불일치
→ 관심사 재검토
→ 연령대 조정

원인 2: 네이버 플레이스 정보 부실
→ 플레이스 최적화 (사진, 리뷰, 소개)

원인 3: 광고와 랜딩페이지 괴리
→ 광고 제목과 플레이스 정보 일치시키기

원인 4: 소재가 오해 유발
→ 명확한 메시지로 수정
```

---

### ❌ 비용이 너무 많이 나가요

```
해결 1: 수동 입찰로 전환
→ 평균 단가의 60~70% 설정

해결 2: 시간대 조정
→ 성과 좋은 시간대만 집행

해결 3: 키워드 정리 (검색 광고)
→ 검색량 많은 일반 키워드 제외
→ 롱테일 키워드 집중

해결 4: 저성과 소재 off
→ CTR 낮은 소재 중지
```

광고 계정:

- [ ] 소재 제목 차별화

최적화 준비:
- [ ] Google Analytics 연동 (선택)
