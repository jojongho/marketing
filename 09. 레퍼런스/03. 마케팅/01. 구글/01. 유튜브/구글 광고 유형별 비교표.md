# 구글 광고 유형별 비교표

> 구글 애즈의 주요 광고 유형을 한눈에 비교

---

## 📊 핵심 비교표

| 항목 | 검색 광고 (Search) | 배너 광고 (GDN) | 유튜브 광고 (디멘드젠) |
|------|-------------------|----------------|---------------------|
| **노출 위치** | 구글 검색 결과 상단 | 구글 제휴 웹사이트/앱 300만+ | 유튜브 인피드, 홈 피드 |
| **광고 형태** | 텍스트 | 이미지 배너 | 영상 + 이미지 |
| **클릭당 비용** | 1,000-2,000원 | 100-200원 | 중간 (500-1,500원) |
| **초기 일 예산 권장** | 2-5만원 | 1-3만원 | 2-5만원 |
| **평균 CTR** | 2-5% | 0.3-1% | 0.5-2% |
| **타게팅 방식** | 키워드 기반 | 관심사, 웹사이트 방문 기록 | 유튜브 검색어, 시청 채널 |
| **전환율** | 높음 (즉시 구매 의도) | 낮음 (인지도 중심) | 중간 (영상 몰입도) |
| **학습 기간** | 1-2주 | 2-3주 | 2-4주 |
| **제작 난이도** | 쉬움 (텍스트만) | 중간 (이미지 제작) | 어려움 (영상 제작) |
| **관리 난이도** | 높음 (검색어 필터링) | 중간 | 중간 |

## 🎯 목적별 추천 광고

### 💰 즉각적인 매출/전환이 필요할 때
**추천: 검색 광고 (Search) ⭐⭐⭐⭐⭐**

**이유:**
- 검색 의도가 명확한 고객 (구매 직전 단계)
- 즉시 전환 가능
- ROI 측정 명확

**적합 업종:**
- 긴급 서비스 (수리, 법률, 의료)
- B2B 서비스
- 부동산 중개
- 고가 상품 (자동차, 가전)

**예산:**
- 최소 일 3만원
- 권장 월 100-300만원

---

### 📢 브랜드 인지도 확산이 목표일 때
**추천: 배너 광고 (GDN) ⭐⭐⭐⭐⭐**

**이유:**
- 저렴한 노출 비용
- 광범위한 도달
- 시각적 임팩트

**적합 업종:**
- 신규 브랜드/제품 런칭
- 이커머스
- 이벤트/프로모션
- 앱 설치 캠페인

**예산:**
- 최소 일 2만원
- 권장 월 60-150만원

---

### 🎬 제품 시연/스토리텔링이 중요할 때
**추천: 유튜브 광고 (디멘드젠) ⭐⭐⭐⭐⭐**

**이유:**
- 영상으로 강력한 메시지 전달
- 높은 몰입도
- 젊은 타겟층 효과적

**적합 업종:**
- 제품 시연 필요 (화장품, 가전, 요리)
- 교육 서비스
- 앱/게임
- 브랜드 스토리텔링

**예산:**
- 최소 일 3만원
- 권장 월 100-200만원

## 💡 상황별 추천 전략

### 🔰 초보자 (처음 광고 시작)
**추천 순서:**
1. **배너 광고(GDN)** 먼저 시작 (저렴, 쉬움)
2. 데이터 수집 후 **검색 광고** 추가
3. 여유 있으면 **유튜브 광고** 테스트

**이유:**
- 배너 광고는 클릭 비용이 저렴해 리스크 낮음
- 관리가 상대적으로 쉬움
- 초기 학습에 적합

---

### 💰 예산 제한적 (월 100만원 이하)
**추천 조합:**
- **배너 광고 70%** (브랜드 인지도)
- **검색 광고 30%** (핵심 키워드만)

**예산 배분 예시 (월 100만원):**
```
배너 광고: 일 2.3만원 (월 70만원)
검색 광고: 일 1만원 (월 30만원)
```

---

### 💼 B2B 서비스
**추천 조합:**
- **검색 광고 80%** (구매 의도 명확)
- **배너 광고 20%** (리타게팅용)

**이유:**
- B2B는 검색을 통한 정보 수집 중심
- 브랜드 인지도보다 전문성 어필 중요
- 검색 광고의 ROI가 더 높음

---

### 🛍️ B2C 이커머스
**추천 조합:**
- **배너 광고 50%** (신규 고객 유입)
- **검색 광고 30%** (구매 의도 고객)
- **유튜브 광고 20%** (제품 시연)

**이유:**
- 다양한 채널 믹스로 고객 여정 전체 커버
- 배너로 브랜드 인지도 → 검색으로 전환
- 유튜브로 제품 매력 강조

## 📈 성장 단계별 전략

### 1단계: 초기 (0-3개월)
**목표:** 데이터 수집, 학습

```yaml
배너 광고:
  예산: 월 60만원
  목표: 브랜드 인지도, 타게팅 테스트

검색 광고:
  예산: 월 40만원
  목표: 핵심 키워드 5-10개만 집중
```

**활동:**
- 주 2-3회 검색어 필터링
- 다양한 타게팅 테스트
- 광고 문구 A/B 테스트

---

### 2단계: 성장 (3-6개월)
**목표:** 효율 최적화, 확장

```yaml
검색 광고:
  예산: 월 100만원 (↑)
  목표: 성과 좋은 키워드 확장

배너 광고:
  예산: 월 80만원 (↑)
  목표: 리타게팅 시작

유튜브 광고:
  예산: 월 50만원 (신규)
  목표: 영상 콘텐츠 테스트
```

**활동:**
- 리타게팅 캠페인 구축
- 전환 추적 고도화
- 자동 규칙 설정

---

### 3단계: 최적화 (6개월 이상)
**목표:** ROI 극대화, 자동화

```yaml
검색 광고:
  예산: 월 200만원 (↑)
  전략: 스마트 입찰 (tCPA, tROAS)

배너 광고:
  예산: 월 150만원 (↑)
  전략: 동적 광고, 리타게팅 강화

유튜브 광고:
  예산: 월 100만원 (↑)
  전략: 영상 시리즈 제작
```

**활동:**
- 자동화 스크립트 활용
- 고급 잠재고객 구축
- 계절별/이벤트별 전략

## ⚖️ 장단점 상세 비교

### 검색 광고 (Search)

#### ✅ 장점
1. **높은 전환율**
   - 구매 의도가 명확한 고객
   - 즉각적인 액션 가능

2. **명확한 ROI**
   - 키워드별 성과 측정 용이
   - 빠른 최적화 가능

3. **정교한 타게팅**
   - 키워드 단위 제어
   - 검색 의도 파악 가능

4. **빠른 효과**
   - 광고 승인 후 즉시 노출
   - 1-2주 내 데이터 충분

#### ❌ 단점
1. **높은 클릭 비용**
   - 경쟁 심화 시 CPC 급등
   - 예산 소진 빠름

2. **관리 부담**
   - 검색어 필터링 필수 (주 2-3회)
   - 키워드 지속 확장 필요

3. **제한적 도달**
   - 검색하는 사람만 노출
   - 잠재 고객 발굴 어려움

4. **브랜드 인지도 효과 낮음**
   - 단기 전환에만 집중
   - 장기적 브랜드 구축 한계

---

### 배너 광고 (GDN)

#### ✅ 장점
1. **저렴한 비용**
   - 클릭당 100-200원
   - 많은 노출 확보 가능

2. **광범위한 도달**
   - 300만+ 웹사이트/앱
   - 잠재 고객 발굴 효과적

3. **시각적 임팩트**
   - 이미지로 강한 인상
   - 브랜드 이미지 구축

4. **리타게팅 강력**
   - 방문자 재유입 효과적
   - 구매 전환율 상승

#### ❌ 단점
1. **낮은 즉시 전환율**
   - 인지도 중심
   - 구매까지 시간 소요

2. **노출 위치 제어 제한**
   - 어떤 사이트에 노출될지 불확실
   - 브랜드 이미지 리스크

3. **배너 피로도**
   - 같은 광고 반복 시 효과 하락
   - 정기적 소재 교체 필요

4. **클릭 품질 편차**
   - 실수 클릭 많음
   - 이탈률 높을 수 있음

---

### 유튜브 광고 (디멘드젠)

#### ✅ 장점
1. **강력한 메시지 전달**
   - 영상으로 제품 시연
   - 감성적 어필 가능

2. **높은 몰입도**
   - 영상 시청 중 집중
   - 브랜드 기억률 높음

3. **정교한 타게팅**
   - 유튜브 시청 기록 기반
   - 관심사 정확히 파악

4. **젊은 층 효과적**
   - 10-40대 주 사용자
   - 트렌드 민감 타겟

#### ❌ 단점
1. **영상 제작 부담**
   - 제작 비용 발생
   - 전문성 필요

2. **학습 기간 긺**
   - 2-4주 데이터 수집 필요
   - 초기 효율 낮을 수 있음

3. **스킵 가능**
   - 일부 광고는 건너뛰기 가능
   - 조회수 ≠ 실제 시청

4. **모바일 중심**
   - PC 효과 상대적으로 낮음
   - 모바일 최적화 필수

## 📊 예산별 추천 전략

### 월 50만원
```yaml
전략: 단일 채널 집중
추천: 배너 광고 (GDN) 100%

이유:
  - 저렴한 비용으로 많은 노출
  - 브랜드 인지도 우선 확보
  - 관리 부담 낮음
```

### 월 100만원
```yaml
전략: 2개 채널 병행
배너 광고: 60만원 (60%)
검색 광고: 40만원 (40%)

이유:
  - 인지도 + 전환 동시 확보
  - 리스크 분산
  - 상호 시너지
```

### 월 200만원
```yaml
전략: 3개 채널 믹스
검색 광고: 80만원 (40%)
배너 광고: 80만원 (40%)
유튜브 광고: 40만원 (20%)

이유:
  - 전체 고객 여정 커버
  - 다양한 접점 확보
  - 최적 채널 발견 가능
```

### 월 500만원 이상
```yaml
전략: 풀 퍼널 (Full Funnel)
검색 광고: 200만원 (40%)
배너 광고: 150만원 (30%)
유튜브 광고: 100만원 (20%)
리타게팅: 50만원 (10%)

이유:
  - 인지도 → 고려 → 전환 전체 최적화
  - 리타게팅으로 전환율 극대화
  - 데이터 기반 자동화 가능
```
