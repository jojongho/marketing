# 구글 광고 세팅 체크리스트

> 광고 세팅 시 빠뜨리지 말아야 할 항목들을 단계별로 정리

---

## 📋 사전 준비 체크리스트

### 계정 및 서류
- [ ] 구글 계정(Gmail) 생성
- [ ] 구글 애즈 계정 생성
- [ ] 사업자등록증 또는 신분증 준비 (광고주 인증용)
- [ ] 결제 수단 등록 (신용카드/체크카드)

### 콘텐츠 준비
- [ ] 광고할 웹사이트/랜딩 페이지 URL
- [ ] 광고 이미지 (1200x1200, 1200x628)
- [ ] 로고 이미지 (300x300)
- [ ] 광고 영상 (유튜브 광고 시)

### 전략 수립
- [ ] 광고 목표 명확화 (매출, 인지도, 리드 등)
- [ ] 타겟 고객 정의 (연령, 성별, 지역, 관심사)
- [ ] 월/일 예산 설정
- [ ] 핵심 키워드 리스트업 (50-100개)

---

## 🔍 검색 광고 체크리스트

### 캠페인 생성
- [ ] 목표: "웹사이트 트래픽" 선택
- [ ] 유형: "검색" 선택
- [ ] 캠페인 이름: 알아보기 쉽게 작성
- [ ] 입찰 전략: "클릭수" 선택
- [ ] 입찰가 한도: 1,000-2,000원 설정

### 네트워크 설정 (중요!)
- [ ] ❌ "구글 검색 파트너" 체크 해제
- [ ] ❌ "구글 디스플레이" 체크 해제

### 위치 및 언어
- [ ] 위치: 타겟 지역 선택 (예: 대한민국, 서울특별시)
- [ ] 언어: 한국어 선택

### 키워드 설정
- [ ] 키워드 50-100개 작성
- [ ] 구문 검색(`"키워드"`) 위주로 작성
- [ ] 일치 검색(`[키워드]`) 일부 혼합
- [ ] 확장 검색은 초기에 제외

### 광고 문구
- [ ] 제목 5개 이상 작성
- [ ] 긴 제목 3개 이상 작성
- [ ] 설명 4개 이상 작성
- [ ] 키워드를 제목/설명에 반복 포함
- [ ] 광고 효력 "좋음" 이상 달성

### 이미지 및 로고
- [ ] 광고 이미지 업로드
- [ ] 비즈니스 로고 업로드

### 예산
- [ ] 일 예산 설정 (2-5만원 권장)
- [ ] "캠페인 게시" 클릭

### 세부 설정 (필수!)
- [ ] 인구 통계 → 나이, 성별 선택
- [ ] ⚠️ "알 수 없음" 항목 해제
- [ ] 사이트 링크 4개 이상 추가 (텍스트 + URL)

### 검색어 필터링 (핵심!)
- [ ] 광고 게시 후 2-3일 대기
- [ ] "통계 및 보고서" → "검색어" 메뉴
- [ ] 관련 없는 검색어 "제외 추가"
- [ ] 주 2-3회 반복 점검

## 🎨 배너 광고(GDN) 체크리스트

### 캠페인 생성
- [ ] 목표: "웹사이트 트래픽" 선택
- [ ] 유형: "디스플레이" 선택
- [ ] 캠페인 이름 작성
- [ ] 위치: 타겟 지역 선택
- [ ] 언어: 한국어 선택

### 예산 및 입찰
- [ ] 일 예산: 1-3만원 설정
- [ ] 입찰: "클릭수 최대화" 선택
- [ ] 입찰가 한도: 100원 설정 (노출 부족 시 120원, 140원으로 조정)

### 광고 소재
- [ ] 이미지 1200x1200 업로드
- [ ] 이미지 1200x628 업로드
- [ ] 로고 300x300 업로드

### 문구
- [ ] 제목 5개 이상 작성
- [ ] 긴 제목 3개 이상 작성
- [ ] 설명 5개 이상 작성
- [ ] 유도 문안(CTA) 선택

### 타게팅 설정 (꿀팁!)
- [ ] "잠재고객 세그먼트 추가" 클릭
- [ ] "맞춤 세그먼트" 생성
- [ ] "검색어를 검색한 사용자" 체크 → 키워드 20-30개 입력
- [ ] "특정 웹사이트 둘러보는 사용자" 체크 → URL 5-10개 입력

### 인구 통계
- [ ] 나이, 성별 선택
- [ ] ⚠️ "알 수 없음" 해제

### 기기 설정
- [ ] "특정 기기" 선택
- [ ] ✅ 컴퓨터 체크
- [ ] ✅ 휴대전화 체크
- [ ] ❌ 태블릿 체크 해제

### 활성화
- [ ] 광고비 충전 ("결제 요약" → "금액 추가")
- [ ] 캠페인 "활성화" 상태로 변경

---

## 📹 유튜브 광고(디멘드젠) 체크리스트

### 영상 준비
- [ ] 광고 영상 제작 (15-30초 권장)
- [ ] 유튜브 채널에 영상 업로드
- [ ] 영상 URL 복사

### 캠페인 생성
- [ ] 목표: "웹사이트 트래픽" 선택
- [ ] 유형: "디맨드젠 캠페인" 선택
- [ ] 캠페인 이름 작성

### 기본 설정
- [ ] 목표: "클릭수" 선택
- [ ] 일 예산: 2-5만원 설정
- [ ] 위치: 타겟 지역 선택
- [ ] 언어: 한국어 선택

### 기기 설정
- [ ] "특정 기기" 선택
- [ ] ✅ 휴대전화만 체크 (유튜브는 모바일 중심)

### 노출 위치
- [ ] 채널: "직접 선택" 클릭
- [ ] ❌ "디스플레이 네트워크" 해제
- [ ] ✅ "유튜브 인피드 광고"만 체크

### 잠재고객 설정 (핵심!)
- [ ] "잠재 고객 추가" → "+ 새 잠재 고객"
- [ ] "맞춤 세그먼트" → "+ 새 세그먼트"
- [ ] "구글에 해당 키워드 검색한 사용자" 선택 → 키워드 20-30개 입력
- [ ] "특정 웹사이트 둘러보는 사용자" 선택 → 경쟁사 URL, 유튜브 영상 URL 10개 입력

### 인구 통계
- [ ] 나이, 성별 선택
- [ ] ⚠️ "알 수 없음" 해제

### 타게팅 최적화 (매우 중요!)
- [ ] ✅ "연령 및 성별 요건에 해당하는 사용자에게만 광고" 체크

### 광고 소재
- [ ] 광고 유형 선택 (영상 광고 권장)
- [ ] 최종 URL 입력
- [ ] 영상 URL 입력
- [ ] "동영상 표시 위치 선택" 체크 → "인피드"만 선택

### 텍스트 및 로고
- [ ] 로고 업로드 (300x300)
- [ ] 광고 제목 5개 이상 작성
- [ ] 긴 광고 제목 3개 이상 작성
- [ ] 설명 5개 이상 작성
- [ ] 유도 버튼 문구 선택
- [ ] 사이트 링크 3개 이상 추가

### 게시
- [ ] "리뷰로 이동" → 오류 확인
- [ ] "캠페인 게시" 클릭

---

## 🔧 공통 필수 설정 체크리스트

### 광고비 및 결제
- [ ] 광고비 충전 완료
- [ ] 결제 정보 등록 (자동 결제 또는 수동 충전)
- [ ] 예산 알림 설정 (선택사항)

### 광고주 인증 (필수!)
- [ ] "결제" → "광고주 인증" 메뉴
- [ ] 사업자등록증 또는 신분증 업로드
- [ ] 승인 대기 (3-5일)

### 모니터링 설정
- [ ] 구글 애즈 앱 설치 (모바일 모니터링)
- [ ] 이메일 알림 설정
- [ ] 주간 성과 보고서 수신 설정

---

## 📊 광고 게시 후 체크리스트

### 첫 주 (데이터 수집)
- [ ] 광고 승인 확인 (1-2 업무일)
- [ ] 광고 노출 확인 (실제 검색해보기)
- [ ] 일일 예산 소진 상황 체크
- [ ] 검색어 보고서 확인 (검색 광고)

### 2주차 (초기 최적화)
- [ ] 검색어 필터링 (관련 없는 키워드 제외)
- [ ] 광고 효력 "좋음" 이상 유지
- [ ] 클릭률(CTR) 확인
- [ ] 입찰가 조정 (노출 부족 시 상향, 예산 초과 시 하향)

### 3-4주차 (본격 최적화)
- [ ] 성과 낮은 광고 일시 중지
- [ ] A/B 테스트 시작 (광고 문구, 타게팅)
- [ ] 타게팅 재조정 (연령, 성별, 지역)
- [ ] 랜딩 페이지 개선 (전환율 낮을 시)

### 1개월 후 (고도화)
- [ ] 전환 추적 설정 (GA4 연동)
- [ ] 리마케팅 광고 고려
- [ ] 광고 일정 설정 (시간대별 차등 입찰)
- [ ] 자동 규칙 설정 (예산 초과 시 자동 일시 중지 등)

---

## ⚠️ 흔한 실수 방지 체크리스트

### 예산 및 입찰
- [ ] 입찰가 한도 설정 없이 "클릭수" 전략 사용 (예산 폭탄 위험)
- [ ] 총 예산 대신 일 예산 사용 확인 (효율성)
- [ ] 일 예산 2배 소진 시 당황하지 않기 (정상 작동)

### 타게팅
- [ ] "알 수 없음" 항목 해제 확인
- [ ] 구글 검색 파트너, 디스플레이 네트워크 해제 확인 (검색 광고)
- [ ] 타게팅 최적화 설정 시 "연령 및 성별 요건" 체크 확인 (유튜브 광고)

### 키워드
- [ ] 확장 검색 남발 방지 (예산 낭비)
- [ ] 검색어 필터링 미실시 방지 (가장 중요!)
- [ ] 키워드 너무 적게 설정 (50개 미만)

### 광고 문구
- [ ] 광고 효력 "나쁨" 상태 방치 (경쟁력 약화)
- [ ] 제목/설명 최소 개수만 작성 (다양성 부족)
- [ ] 키워드와 무관한 문구 작성

### 기타
- [ ] 광고주 인증 미실시 (계정 정지 위험)
- [ ] 광고 게시 후 방치 (최적화 없음)
- [ ] 데이터 수집 전 조기 판단 (최소 2주 필요)

## 🎯 성과 측정 체크리스트

### 주요 지표
- [ ] 노출수 (Impressions)
- [ ] 클릭수 (Clicks)
- [ ] 클릭률 (CTR) - 목표: 검색 광고 2% 이상, 배너 광고 0.5% 이상
- [ ] 평균 클릭 비용 (CPC)
- [ ] 전환수 (Conversions) - 전환 추적 설정 후
- [ ] 전환율 (CVR)
- [ ] 광고 투자 수익률 (ROAS)

### 주간 점검 사항
- [ ] 예산 소진 속도
- [ ] 검색어 보고서 (새로운 제외 키워드)
- [ ] 광고 효력 유지 상태
- [ ] 경쟁사 광고 모니터링

### 월간 점검 사항
- [ ] 목표 대비 성과 달성률
- [ ] 캠페인별 ROI 분석
- [ ] 타게팅 전략 재조정
- [ ] 예산 재배분 (성과 좋은 캠페인에 집중)

## 📌 빠른 참조 요약

### 🚫 절대 하지 말 것
- ❌ 검색 광고에서 "구글 검색 파트너" 체크 유지
- ❌ "알 수 없음" 타게팅 포함
- ❌ 검색어 필터링 하지 않기
- ❌ 광고주 인증 미실시
- ❌ 광고 효력 "나쁨" 방치

### ✅ 반드시 할 것
- ✅ 검색어 필터링 (주 2-3회)
- ✅ 광고 효력 "좋음" 이상 유지
- ✅ 입찰가 한도 설정 (예산 통제)
- ✅ 사이트 링크 추가 (클릭률 향상)
- ✅ 광고주 인증 완료 (계정 보호)
