## 시작 전 준비사항

### ✅ 필수 준비물
- [ ] 구글 계정 (Gmail)
- [ ] 광고비 결제 수단 (신용카드/체크카드)
- [ ] 광고할 웹사이트 또는 랜딩 페이지 URL
- [ ] 사업자등록증 (광고주 인증용)

### 💰 초기 예산 권장사항
- **검색 광고**: 일 예산 2-5만원 (클릭당 1,000-2,000원)
- **배너 광고(GDN)**: 일 예산 1-3만원 (클릭당 100-200원)
- **유튜브 광고**: 일 예산 2-5만원 (클릭수 기준)

**⚠️ 주의:** 구글은 일 예산의 2배까지 사용할 수 있으나, 장기적으로는 평균 일 예산에 맞춰집니다.

### 📌 구글 애즈 계정 생성
1. https://ads.google.com 접속
2. "지금 시작하기" 클릭
3. Gmail 계정으로 로그인
4. 초기 설정은 건너뛰기 (직접 세팅할 예정)

## 구글 광고 유형 이해하기

### 1️⃣ 검색 광고 (Search Ads)
**📍 노출 위치:** 구글 검색 결과 상단

**✅ 장점:**
- 검색 의도가 명확한 잠재고객 타게팅
- 즉각적인 전환 가능
- 키워드 기반 정교한 타게팅

**❌ 단점:**
- 클릭당 비용이 상대적으로 높음 (1,000-2,000원)
- 키워드 경쟁 심화

**👤 추천 대상:**
- 즉각적인 매출/전환이 목표
- 특정 키워드 검색자를 타게팅하고 싶을 때
- B2B 서비스, 긴급 서비스 (예: 변호사, 수리업체)

---

### 2️⃣ 배너 광고 / GDN (Display Network)
**📍 노출 위치:** 구글 제휴 300만+ 웹사이트, 앱

**✅ 장점:**
- 저렴한 클릭 비용 (100-200원)
- 광범위한 노출
- 브랜드 인지도 상승

**❌ 단점:**
- 즉각적인 전환율 낮음
- 노출 위치 컨트롤 제한적

**👤 추천 대상:**
- 브랜드 인지도 확산
- 예산이 제한적일 때
- B2C 제품/서비스

---

### 3️⃣ 유튜브 광고 / 디멘드젠 (Demand Gen)
**📍 노출 위치:** 유튜브 인피드, 홈 피드

**✅ 장점:**
- 유튜브에 집중적으로 노출
- 영상 콘텐츠로 강력한 메시지 전달
- 정교한 잠재고객 타게팅 (검색어, 채널 기반)

**❌ 단점:**
- 영상 제작 필요
- 초기 학습 기간 필요

**👤 추천 대상:**
- 제품 시연이 필요한 상품
- 브랜드 스토리텔링 중요
- 젊은 타겟층 (유튜브 사용자)

---

## 구글 검색 광고 세팅

### Step 1: 캠페인 생성
1. 구글 애즈 로그인 → 왼쪽 상단 **"+ 만들기"** → **"캠페인"**
2. 목표 선택: **"웹사이트 트래픽"**
3. 유형 선택: **"검색"**
4. 웹사이트 URL 입력 → "계속"

### Step 2: 캠페인 설정
```yaml
캠페인 이름: 알아보기 쉬운 이름 (예: "서울_부동산_검색광고")
입찰 전략: "클릭수"로 변경
입찰가 한도: 1,000원 ~ 2,000원
```

**⚠️ 중요 설정:**
- ❌ **구글 검색 파트너** 체크 해제
- ❌ **구글 디스플레이** 체크 해제
→ 이유: 구글 검색에만 노출시켜 효율 극대화

### Step 3: 위치 및 언어
- 위치: 대한민국 (또는 원하는 지역)
- 언어: 한국어

### Step 4: 키워드 설정 (가장 중요!)

#### 키워드 매칭 유형 3가지

| 유형 | 표기법 | 노출 범위 | 예시 |
|------|--------|-----------|------|
| **확장 검색** | `가구 매장` | 포괄적 (가장 넓음) | "인테리어 가구", "가죽 소파" 등 |
| **구문 검색** ✅ | `"가구 매장"` | 중간 (키워드 중심) | "저렴한 가구 매장", "침대 가구 매장" |
| **일치 검색** | `[가구 매장]` | 정확 (가장 좁음) | "가구 매장" 정확히 검색 시만 |

**🎯 초보자 권장 전략:**
- **구문 검색(`"키워드"`)** 위주로 50-100개
- 일치 검색 일부 혼합
- 확장 검색은 처음엔 피하기 (예산 낭비 가능)

**📝 키워드 작성 예시:**
```
"강남 부동산"
"강남 아파트 매매"
"강남역 오피스텔"
[강남 부동산]
[강남 아파트]
```

### Step 5: 광고 문구 작성

```yaml
제목: 최소 3개 이상
  - "강남 아파트 실시간 매물"
  - "중개수수료 최대 50% 할인"
  - "10년 경력 전문가 상담"

설명: 최소 2개 이상
  - "강남 아파트 실거래가 분석 및 투자 상담. 지금 바로 무료 상담받으세요!"
  - "빠른 매물 등록, 신속한 계약. 전문가가 도와드립니다."
```

**✨ 광고 효력 "좋음" 이상 맞추기:**
- 키워드를 제목과 설명에 반복 포함
- 제목 5개 이상, 설명 4개 이상 작성
- 다양한 각도의 메시지 (가격, 품질, 서비스 등)

### Step 6: 이미지 및 로고
- 광고 이미지 업로드
- 비즈니스 로고 업로드

### Step 7: 예산 설정
- 일 예산: 2-5만원 (초기)
- "캠페인 게시" 클릭

### Step 8: 세부 설정 (필수!)

#### 8-1. 인구 통계 설정
1. 좌측 메뉴 → **"캠페인"** → **"잠재고객, 키워드, 콘텐츠"**
2. **"인구 통계 수정"** 클릭
3. 원하는 나이, 성별 선택
4. ⚠️ **"알 수 없음" 반드시 해제** (정확도 향상)

#### 8-2. 사이트 링크 추가
1. 좌측 메뉴 → **"캠페인"** → **"애셋"**
2. **"사이트 링크"** 선택 → "+" 버튼
3. 링크 텍스트 + URL 입력 (4개 이상 권장)

**예시:**
```
- "빠른 상담 신청" → https://example.com/contact
- "실거래가 확인" → https://example.com/price
- "매물 등록하기" → https://example.com/register
- "회사 소개" → https://example.com/about
```

### Step 9: 🔥 검색어 필터링 (핵심!)

> **광고 효율을 좌우하는 가장 중요한 작업**

1. 좌측 메뉴 → **"통계 및 보고서"** → **"검색어"**
2. 광고 노출 후 데이터 확인
3. **관련 없는 검색어 체크** → **"제외 추가"** 클릭

**📌 필터링 주기:**
- 초기 1주일: 매일 체크
- 이후: 주 2-3회

## 구글 배너 광고(GDN) 세팅

### Step 1: 캠페인 생성
1. "+ 만들기" → "캠페인"
2. 목표: **"웹사이트 트래픽"**
3. 유형: **"디스플레이"**
4. URL 입력 → "계속"

### Step 2: 캠페인 설정
```yaml
캠페인 이름: "GDN_배너광고"
위치: 대한민국
언어: 한국어
일 예산: 1-3만원
입찰: "클릭수 최대화" 선택
입찰가 한도: 100원 (노출 부족 시 120원, 140원으로 상향)
```

### Step 3: 광고 소재 준비 (필수!)

**📐 필요한 이미지 사이즈:**
- 1200 x 1200 (정사각형)
- 1200 x 628 (와이드)
- 로고: 300 x 300

**🎨 이미지 제작 팁:**
- 명확한 메시지 (3초 안에 이해 가능)
- 강한 CTA (예: "지금 신청", "무료 상담")
- 브랜드 로고 명확히 표시

### Step 4: 문구 작성
```yaml
제목: 최대한 많이 작성 (5개 이상)
  - "강남 부동산 전문"
  - "실시간 매물 확인"
  - "중개수수료 50% 할인"

긴 제목: 3개 이상
  - "강남 아파트 실거래가 분석 전문 부동산"

설명: 5개 이상
  - "10년 경력 전문가가 직접 상담해드립니다"
  - "빠른 매물 등록, 신속한 계약 진행"
```

### Step 5: 🎯 타게팅 설정 (꿀팁!)

#### 5-1. 맞춤 세그먼트 생성
1. 좌측 메뉴 → **"잠재고객"**
2. **"잠재고객 세그먼트 추가"** → **"맞춤 세그먼트"**
3. **"+ 맞춤"** 클릭

#### 5-2. 타게팅 전략 (2가지 방법)

**방법 1: 검색어 기반**
- **"검색어를 검색한 사용자"** 체크
- 관련 키워드 20-30개 입력

```
강남 부동산
강남 아파트
강남역 오피스텔
서초동 빌라
압구정 부동산
...
```

**방법 2: 특정 웹사이트/영상 본 사용자**
- **"특정 유형의 웹사이트를 둘러보는 사용자"** 클릭
- 경쟁사 홈페이지 URL 5-10개 입력
- 조회수 높은 관련 유튜브 영상 URL 5-10개 입력

### Step 6: 인구 통계 설정
- 원하는 나이, 성별 선택
- ⚠️ **"알 수 없음" 해제** 필수

### Step 7: 기기 설정
1. 캠페인 → 톱니바퀴 → "추가" → "기기"
2. **"특정 기기"** 선택
3. ✅ 컴퓨터, ✅ 휴대전화 (태블릿은 보통 제외)

### Step 8: 광고비 충전 및 활성화
1. 좌측 메뉴 → **"결제 요약"** → **"금액 추가"**
2. 원하는 금액 충전
3. 캠페인 → **"활성화"** 상태로 변경

## 유튜브 광고(디멘드젠) 세팅

### Step 1: 영상 준비 (필수!)
- 유튜브 채널에 광고 영상 업로드
- 권장 길이: 15-30초 (짧고 임팩트 있게)
- 첫 3초에 핵심 메시지

### Step 2: 캠페인 생성
1. "+ 만들기" → "캠페인"
2. 목표: **"웹사이트 트래픽"**
3. 유형: **"디맨드젠 캠페인"**

### Step 3: 캠페인 설정
```yaml
캠페인 이름: "디멘드젠_유튜브광고"
목표: "클릭수" (전환 스크립트 없을 때)
일 예산: 2-5만원
위치: 대한민국
언어: 한국어
```

### Step 4: 기기 설정
- **"특정 기기"** 선택
- ✅ **휴대전화만 체크** (유튜브는 모바일 사용 90% 이상)

### Step 5: 노출 위치 설정
1. 채널 → **"직접 선택"**
2. ❌ **"디스플레이 네트워크" 해제**
3. ✅ **"유튜브 인피드 광고"만 체크** (가장 효과적)

### Step 6: 🎯 잠재고객 설정 (핵심!)

#### 6-1. 맞춤 세그먼트 생성
1. **"잠재 고객 추가"** → **"+ 새 잠재 고객"**
2. **"맞춤 세그먼트"** → **"+ 새 세그먼트"**

#### 6-2. 타게팅 전략 (2가지 결합)

**전략 1: 구글 검색 키워드 기반**
- **"구글에 해당 키워드를 검색한 사용자"** 선택
- 관련 키워드 20-30개 입력

**전략 2: 특정 채널/영상 시청자**
- **"특정 유형의 웹사이트를 둘러보는 사용자"** 클릭
- 경쟁사 홈페이지 URL 입력
- 조회수 높은 관련 유튜브 영상 URL 10개 입력

```
예시:
- https://www.youtube.com/watch?v=xxxxx (부동산 투자 인기 영상)
- https://www.youtube.com/watch?v=yyyyy (강남 아파트 리뷰)
- 경쟁사 홈페이지 URL
```

### Step 7: 인구 통계 설정
- 나이, 성별 선택
- ⚠️ **"알 수 없음" 해제**

### Step 8: 타게팅 최적화 (매우 중요!)
1. 하단 **"타게팅 최적화"** 섹션
2. ✅ **"연령 및 성별 요건에 해당하는 사용자에게만 광고"** 체크

→ 이것을 체크하지 않으면 구글이 자동으로 타게팅 변경!

### Step 9: 광고 소재 설정

#### 9-1. 광고 유형 선택
- 단일 이미지 광고
- 영상 광고 (권장)
- 캐러셀 이미지 광고

#### 9-2. URL 및 영상
```yaml
최종 URL: 랜딩 페이지 URL
영상: 유튜브 업로드한 영상 URL
```

#### 9-3. 동영상 표시 위치
- ✅ **"동영상 표시 위치 선택"** 체크
- **"모든 형식"** 클릭 → **"인피드"만 체크**

#### 9-4. 로고 및 텍스트
```yaml
로고: 300x300 업로드

광고 제목 (5개 이상):
  - "강남 부동산 전문가 상담"
  - "실시간 아파트 매물 확인"
  - "중개수수료 최대 50% 할인"

긴 광고 제목 (3개 이상):
  - "강남 아파트 실거래가 분석 및 투자 상담"

설명 (5개 이상):
  - "10년 경력 전문가가 직접 상담해드립니다"
  - "빠른 매물 확인, 신속한 계약 진행"
```

#### 9-5. 유도 버튼 문구
- 목적에 맞는 CTA 선택 (예: "자세히 알아보기", "지금 신청")

#### 9-6. 사이트 링크
```
- "무료 상담 신청" → URL
- "실거래가 확인" → URL
- "매물 등록" → URL
```

### Step 10: 캠페인 게시
1. **"리뷰로 이동"**
2. 빨간색 오류 없는지 확인
3. **"캠페인 게시"**

## 광고 효율 높이는 핵심 팁

### 1️⃣ 검색어 필터링 (검색 광고)

**중요도: ⭐⭐⭐⭐⭐**

> 광고비 낭비를 막는 가장 중요한 작업!

**실행 방법:**
1. 캠페인 게시 후 2-3일 대기 (데이터 축적)
2. "통계 및 보고서" → "검색어" 메뉴
3. 내 사업과 관련 없는 키워드 체크 → "제외 추가"

**필터링 예시:**
```
광고 키워드: "영어 그림책"

관련 없는 검색어 (제외 대상):
❌ "무료 영어 그림책"
❌ "영어 그림책 PDF"
❌ "영어 그림책 다운로드"
```

**주기:**
- 초기 1주: 매일
- 1주 후: 주 2-3회
- 1개월 후: 주 1회

---

### 2️⃣ 광고 효력(Ad Strength) "좋음" 이상 유지

**현재 효력 확인:**
- 광고 작성 화면 상단에 표시
- "나쁨", "평균", "좋음", "매우 좋음"

**"좋음" 만드는 방법:**
✅ 제목 5개 이상 작성
✅ 설명 4개 이상 작성
✅ 키워드를 제목/설명에 반복 포함
✅ 다양한 각도의 메시지 (가격, 품질, 속도, 전문성 등)

---

### 3️⃣ 타게팅 정밀화

**인구 통계:**
- ⚠️ **"알 수 없음" 항목 반드시 해제**
- 명확한 타겟층만 선택

**지역:**
- 광범위한 지역보다 **구체적인 지역** 선택
- 예: "대한민국" → "서울특별시 강남구"

**기기:**
- 모든 기기보다 **특정 기기** 선택
- 대부분의 경우 **휴대전화 + 컴퓨터**
- 태블릿은 보통 제외

---

### 4️⃣ 입찰 전략

**초보자 권장:**
- "클릭수 최대화" 선택
- 입찰가 한도 설정 (예산 통제)

**입찰가 한도 설정:**
```yaml
검색 광고: 1,000-2,000원
배너 광고(GDN): 100-200원
유튜브 광고: 시작 시 한도 설정 X (데이터 수집 후 조정)
```

**조정 전략:**
- 노출 부족 → 입찰가 20-30% 상향
- 예산 초과 → 입찰가 10-20% 하향

---

### 5️⃣ A/B 테스트

**테스트 항목:**
- 광고 문구 (제목, 설명)
- 이미지/영상
- 타게팅 (연령, 성별, 지역)
- 랜딩 페이지

**방법:**
- 동일 캠페인 내 광고 2개 이상 생성
- 1-2주 데이터 수집
- 성과 낮은 광고 일시 중지

---

### 6️⃣ 광고 일정 설정

**시간대별 전략:**
- B2C: 평일 저녁 (18-22시), 주말 오후
- B2B: 평일 업무시간 (09-18시)

**설정 방법:**
1. 캠페인 → 톱니바퀴 → "추가" → "광고 일정"
2. 원하는 요일/시간 선택

---

### 7️⃣ 전환 추적 설정 (중급)

**설치 시점:**
- 광고 시작 2-4주 후 (기본 데이터 수집 후)

**추적 항목:**
- 구매 완료
- 상담 신청
- 전화 클릭
- 페이지 체류 시간

## 자주 묻는 질문(FAQ)

### Q1. 광고 승인은 얼마나 걸리나요?
**A:** 일반적으로 1-2 업무일. 빠르면 몇 시간 내 승인.

⚠️ **거부 사유:**
- 금지된 콘텐츠 (도박, 성인물 등)
- 오해의 소지가 있는 문구
- 랜딩 페이지 품질 낮음

---

### Q2. 광고비는 얼마부터 시작해야 하나요?
**A:**
- **최소 일 예산:** 1-2만원
- **권장 초기 예산:** 일 3-5만원 (1개월: 90-150만원)
- **테스트 기간:** 최소 2-4주 (학습 기간)

---

### Q3. 광고 효과는 언제부터 나타나나요?
**A:**
- **검색 광고:** 즉시 (클릭 즉시 유입)
- **배너/유튜브 광고:** 1-2주 후 (브랜드 인지도 상승 필요)

---

### Q4. 네이버 vs 구글 어떤 게 더 좋나요?
**A:**

| 항목 | 네이버 | 구글 |
|------|--------|------|
| 검색 점유율 | 60% | 30% |
| 클릭당 비용 | 높음 (2,000-5,000원) | 중간 (1,000-2,000원) |
| 타게팅 정교함 | 중간 | 높음 |
| 광고 유형 다양성 | 중간 | 높음 (YouTube 등) |

**결론:** 둘 다 병행 추천 (예산 허용 시)

---

### Q5. 광고주 인증은 꼭 해야 하나요?
**A:** ✅ **필수**

- 미인증 시 계정 정지 가능
- 사업자등록증 또는 신분증 필요
- 승인까지 3-5일 소요

**인증 방법:**
1. 좌측 메뉴 → **"결제"** → **"광고주 인증"**
2. 서류 업로드 → 승인 대기

---

### Q6. 일 예산을 초과해서 광고비가 나갔어요!
**A:** 정상입니다.

**구글 특성:**
- 일 예산의 최대 2배까지 사용 가능
- 하지만 **월 평균은 일 예산 × 30일**에 맞춰짐

**예시:**
- 일 예산 20,000원 설정
- 어느 날 40,000원 사용
- 다음 날 10,000원 사용
- 월 말에는 평균 20,000원/일로 조정됨

---

### Q7. 광고가 노출되지 않아요!
**A:** 체크리스트:

- [ ] 광고 승인 완료되었는가?
- [ ] 광고비가 충전되어 있는가?
- [ ] 캠페인이 "활성화" 상태인가? (일시 중지 아닌지)
- [ ] 입찰가가 너무 낮지 않은가? (경쟁 입찰에서 밀림)
- [ ] 타게팅이 너무 좁지 않은가? (연령, 지역 등)

---

### Q8. 클릭은 많은데 전환이 없어요!
**A:** 원인 및 해결책:

**1. 랜딩 페이지 문제**
- 페이지 로딩 속도 느림
- 모바일 최적화 안 됨
- CTA 버튼 불명확

**2. 타게팅 문제**
- 관련 없는 키워드 필터링 필요
- 타겟 연령/지역 재조정

**3. 광고-랜딩 페이지 불일치**
- 광고 메시지 ≠ 랜딩 페이지 내용
- 일관성 있게 수정

---

### Q9. 구글 vs 네이버 광고를 병행하면 어떤가요?
**A:** ✅ **권장**

**병행 시 장점:**
- 검색 시장 90% 이상 커버
- 리스크 분산
- 상호 보완 (네이버: 즉시성, 구글: 타게팅)

**예산 배분 예시:**
- 월 300만원 → 네이버 60% (180만원), 구글 40% (120만원)
- 성과 데이터 보고 조정

---

### Q10. 광고 대행사를 써야 하나요?
**A:** 상황에 따라 다름

**직접 운영 추천:**
- 예산 월 100만원 이하
- 단순한 서비스/제품
- 마케팅 학습 목적

**대행사 추천:**
- 예산 월 500만원 이상
- 복잡한 타게팅 필요
- 시간 부족
- 전문성 필요 (쇼핑몰, 금융 등)

**대행 수수료:** 통상 광고비의 10-20%
