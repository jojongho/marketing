# 네이버 파워링크 확장소재 노출 제한 해결 방법

## 🔍 문제 상황

### 발생한 문제
- **이미지형 서브링크** 첫 번째에 부동산 매장 사진 등록
- **노출 제한** 발생: "광고소재 노출 제한"
- **제한 사유:** 전화번호 또는 과도한 텍스트 포함

### 원인 분석
네이버 파워링크는 확장소재(이미지형 서브링크)에 대해 **엄격한 정책**을 적용합니다:
- 전화번호가 이미지에 포함되면 반려
- 과도한 텍스트(광고 문구 등)가 포함되면 반려
- 매장 사진에 간판, 전화번호, 홍보 문구가 보이면 반려 가능

---

## ✅ 해결 방법

### 방법 1: 이미지 수정 (권장)

#### 원본 이미지에서 제거해야 할 요소
1. **전화번호** 완전 제거
2. **과도한 텍스트** 제거 (광고 문구, 홍보 문구 등)
3. **간판의 전화번호 부분** 제거 또는 모자이크

#### 권장 이미지 형태
- **깔끔한 매장 외관**만 보이도록
- **간판의 상호명만** 보이도록 (전화번호 제외)
- **최소한의 텍스트**만 포함

#### 이미지 편집 방법
1. **포토샵/피그마** 사용
   - 전화번호 부분 제거 또는 모자이크
   - 텍스트 오버레이 제거
2. **온라인 편집 도구** 사용
   - Photopea (무료 포토샵 대체)
   - Canva (간단한 편집)
   - Remove.bg (배경 제거 후 재구성)

---

### 방법 2: 다른 소재로 대체

#### 대체 이미지 옵션

**옵션 1: 단지 외관 사진**
- 매장 사진 대신 단지 외관 사진 사용
- 전화번호나 텍스트가 없는 깔끔한 이미지

**옵션 2: 단지 조감도**
- 단지 배치도 또는 조감도 사용
- 텍스트가 최소화된 이미지

**옵션 3: 커뮤니티 시설 사진**
- 수영장, 스포츠센터 등 시설 사진
- 매장 사진보다 반려 가능성 낮음

---

### 방법 3: 네이버 플레이스/블로그 활용 (사용자 추측 확인)

#### 네이버 플레이스 활용
1. **네이버 플레이스에 매장 사진 등록**
   - 네이버 플레이스는 전화번호 표시 허용
   - 플레이스 사진을 확장소재로 연결 가능

2. **확장소재 연결 방법**
   - 확장소재 등록 시 "네이버 플레이스" 선택
   - 플레이스에 등록된 사진 자동 연결
   - 플레이스 사진은 정책 검수 통과 가능성 높음

#### 네이버 블로그 활용
1. **블로그 포스트 작성**
   - 매장 소개 포스트 작성
   - 사진에 전화번호 포함 가능 (블로그는 허용)
   
2. **확장소재 연결**
   - 확장소재 등록 시 "블로그 포스트" 선택
   - 블로그 포스트의 이미지 연결

#### 네이버페이 부동산 활용
1. **네이버페이 부동산 등록**
   - 부동산 매물 등록 시 사진 업로드
   - 전화번호 포함 가능
   
2. **확장소재 연결**
   - 네이버페이 부동산의 사진을 확장소재로 연결

## 🔧 실전 해결 단계

### Step 1: 현재 이미지 분석
1. **이미지 확인**
   - 전화번호가 보이는지 확인
   - 과도한 텍스트가 있는지 확인
   - 간판의 홍보 문구 확인

2. **제한 사유 확인**
   - 파워링크 관리자 센터에서 정확한 제한 사유 확인
   - "전화번호 포함" 또는 "과도한 텍스트" 확인

### Step 2: 이미지 수정 또는 대체

#### 옵션 A: 이미지 수정
1. 원본 이미지 다운로드
2. 이미지 편집 도구로 전화번호 제거
3. 텍스트 최소화
4. 수정된 이미지 업로드

#### 옵션 B: 다른 이미지 사용
1. 단지 외관 사진 준비
2. 전화번호/텍스트 없는 깔끔한 이미지 선택
3. 새 이미지로 교체

#### 옵션 C: 네이버 플레이스 활용
1. 네이버 플레이스에 매장 사진 등록
2. 확장소재 등록 시 "플레이스 연결" 선택
3. 플레이스 사진 자동 연결

### Step 3: 재등록 및 검수 대기
1. 수정된 이미지 또는 대체 이미지 업로드
2. 확장소재 재등록
3. 검수 대기 (보통 1~2일)
4. 노출 확인

## 📋 네이버 파워링크 확장소재 정책

### 허용되는 이미지
- ✅ 깔끔한 매장 외관 (전화번호 제외)
- ✅ 상품/서비스 사진
- ✅ 단지 외관, 조감도
- ✅ 커뮤니티 시설 사진
- ✅ 최소한의 텍스트만 포함

### 제한되는 이미지
- ❌ 전화번호가 보이는 이미지
- ❌ 과도한 텍스트 (광고 문구, 홍보 문구)
- ❌ 간판의 전화번호 부분
- ❌ QR코드와 전화번호가 함께 있는 이미지
- ❌ 할인/이벤트 문구가 과도한 이미지

### 텍스트 가이드라인
- **최소한의 텍스트만:** 상호명 정도만 허용
- **전화번호 금지:** 어떤 형태로든 전화번호 표시 불가
- **홍보 문구 금지:** "특가", "할인", "이벤트" 등 과도한 문구 금지

## 🎯 권장 전략

### 전략 1: 이미지형 서브링크 구성 (부동산 업종)

#### 서브링크 1: 단지 정보
- **이미지:** 단지 외관 또는 조감도
- **텍스트:** 전화번호 없이 깔끔한 이미지
- **연결 URL:** 단지 정보 페이지

#### 서브링크 2: 단지 배치도
- **이미지:** 단지 배치도 (텍스트 최소화)
- **연결 URL:** 배치도 상세 페이지

#### 서브링크 3: 위치/오시는 길
- **이미지:** 위치 지도 또는 교통편 안내
- **연결 URL:** 위치 안내 페이지

### 전략 2: 네이버 플레이스 연동 활용

#### 장점
- 네이버 플레이스는 전화번호 표시 허용
- 플레이스 사진을 확장소재로 연결 가능
- 정책 검수 통과 가능성 높음

#### 활용 방법
1. 네이버 플레이스에 매장 사진 등록 (전화번호 포함 가능)
2. 확장소재 등록 시 "플레이스 연결" 선택
3. 플레이스 사진 자동 연결

## 🔍 이미지 URL 추적 방법

### 방법 1: 크롬 개발자도구로 curl 복사

#### 단계별 가이드
1. **개발자도구 열기**
   - `F12` 또는 `Cmd+Option+I` (Mac) / `Ctrl+Shift+I` (Windows)

2. **네트워크 탭 열기**
   - "Network" 탭 클릭
   - 이미지 필터 선택 (Img)

3. **이미지 요청 찾기**
   - 페이지 새로고침
   - 네트워크 목록에서 해당 이미지 URL 찾기
   - `searchad-phinf.pstatic.net` 도메인 확인

4. **curl 복사**
   - 요청 항목 우클릭
   - "Copy" → "Copy as cURL" 선택

5. **curl 실행하여 헤더 확인**
   ```bash
   curl 'https://searchad-phinf.pstatic.net/...' \
     -H 'referer: https://search.naver.com/' \
     -v
   ```
   - `referer` 헤더에서 어느 페이지에서 요청되었는지 확인 가능

### 방법 2: 네이버 파워링크 관리자 센터에서 확인

1. **파워링크 관리자 센터 접속**
2. **확장소재 관리** 메뉴
3. **이미지형 서브링크** 목록 확인
4. **각 서브링크의 이미지 URL** 확인
5. **URL 비교**하여 매칭되는 소재 확인

### 방법 3: 네이버 검색 결과에서 확인
1. **네이버 검색**에서 해당 키워드 검색
2. **파워링크 광고** 확인
3. **확장소재 노출** 확인
4. **이미지 우클릭** → "이미지 주소 복사"
5. **URL 비교**하여 어느 소재인지 확인

## 💡 추가 팁

### 팁 1: 이미지 최적화
- **해상도:** 160x160px 이상 권장
- **파일 크기:** 200KB 이하 권장
- **형식:** JPG 또는 PNG
- **품질:** 고품질이지만 파일 크기는 작게

### 팁 2: A/B 테스트
- 여러 이미지 버전 준비
- 전화번호 제거 버전, 텍스트 최소화 버전 등
- 각각 등록하여 검수 결과 비교

### 팁 3: 네이버 플레이스 우선 활용
- 부동산 업종은 네이버 플레이스 필수
- 플레이스 사진을 확장소재로 연결하면 정책 위반 가능성 낮음
- 전화번호는 플레이스에 표시하고, 확장소재 이미지에는 제외
