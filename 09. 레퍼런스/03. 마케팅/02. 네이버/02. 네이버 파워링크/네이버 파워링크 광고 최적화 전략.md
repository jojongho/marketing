# 네이버 파워링크 광고 최적화 전략

## 🎯 Overview

**핵심 질문**: iusell.cheonan-asan.com 사이트로 어떻게 가장 효율적인 파워링크 광고를 운영할 것인가?

**목표**:
- 최소 비용으로 최대 전환율 달성
- 부동산 중개 특성에 맞는 키워드 전략 수립
- 데이터 기반 광고 운영 시스템 구축

**진행 현황**: 🟡 Framing 단계 → Logging 준비 중

---

## 🔲 Framing

### 왜 이 주제를 탐구하는가?

- **경쟁 우위 확보**: 천안/아산 부동산 시장에서 온라인 우위 선점 필요
- **투자 대비 효율**: 광고비를 들이는 만큼 확실한 성과 창출 필요
- **체계적 운영**: 감이 아닌 데이터 기반의 과학적 광고 운영 시스템 구축

### 핵심 질문

1. **키워드 전략**: 천안/아산 부동산에 최적화된 키워드 조합은 무엇인가?
   - 지역 키워드 (천안아파트, 아산신축, 천안역세권 등)
   - 유형 키워드 (분양권, 전세, 투자용 등)
   - 롱테일 vs 빅키워드 비율은?

2. **입찰 전략**: 한정된 예산으로 최대 효율을 내는 입찰가 설정은?
   - PC vs 모바일 입찰가 차별화
   - 시간대/요일별 입찰 조정
   - 품질지수 향상으로 입찰가 효율화

3. **전환 최적화**: 클릭을 실제 문의/매출로 연결하는 방법은?
   - 랜딩페이지 최적화
   - 전환추적 설정 및 분석
   - A/B 테스트 전략

### 탐구 범위

**포함**:
- 네이버 검색광고 파워링크 전문 운영법
- 부동산 업종 특화 키워드 전략
- 입찰가 최적화 및 예산 관리
- 전환추적 및 성과 분석
- 광고 소재(문구) 최적화
- 확장소재 활용 전략

**제외**:
- 디스플레이 광고 (배너광고)
- 브랜드 검색 광고
- 쇼핑 검색 광고
- SNS 광고 (네이버 외 플랫폼)

## ⚙️ Optimization - 개념 발전

```dataview
LIST
FROM ""
WHERE contains(related_flow, this.file.link)
  AND contains(tags, "개념")
SORT file.mtime DESC
```

### 핵심 인사이트

**광고 구조의 이해**:
- 캠페인 → 광고그룹 → 키워드+소재 의 3단계 구조
- PC/모바일 분리 운영으로 효율 극대화
- 디바이스별 전환율 차이 활용

**키워드 전략**:
- '쓸모있는 키워드'를 '최대한 많이' 확보하는 것이 성공 핵심
- 빅키워드(경쟁 심화) + 롱테일(저비용 고효율) 균형
- 경쟁사가 놓친 틈새 키워드 발굴

**입찰가 최적화**:
- 품질지수가 높을수록 같은 입찰가로 더 높은 순위
- 키워드별 효율 등급 설정 후 차등 입찰
- 비효율 키워드는 과감히 입찰가 하향

- [ ] [[부동산 고객 여정별 키워드 맵]]

### 문제점 및 개선 방향

**예상 문제점**:
1. 초기 데이터 부족으로 최적 입찰가 불확실
2. 부동산 업종 특성상 전환까지 긴 여정
3. 경쟁사 많은 지역 키워드의 높은 CPC

**개선 방향**:
1. 소액 예산으로 테스트 기간 운영 → 데이터 축적
2. 중간 전환 설정 (전화클릭, 채팅시작 등)
3. 롱테일 키워드 공략 + 확장소재로 노출면적 확대
