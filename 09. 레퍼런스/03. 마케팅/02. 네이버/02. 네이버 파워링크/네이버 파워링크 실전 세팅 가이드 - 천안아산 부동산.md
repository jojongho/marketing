## 🔧 STEP 2: 키워드 조합 (5,000개 이상 생성)
### 2-1. 키워드 조합기 사용
1. **사이트 접속**: 네이버에서 "키워드 조합기" 검색
   - 추천: 아이보스 키워드 조합기 또는 키워드 사운드

2. **키워드 입력:**
   - **키워드 1**: 지역 키워드 (천안역, 아산역, 두정동, 배방읍 등)
   - **키워드 2**: 수식어 (근처, 주변)
   - **키워드 3**: 거래유형 (아파트, 전세, 매매, 분양 등)
   - **키워드 4**: 조합 키워드 (부동산, 급매, 추천, 시세 등)

3. **조합 설정:**
   ```
   1-2-3 (천안역 근처 아파트)
   1-2-4 (천안역 근처 부동산)
   1-3 (천안역 아파트)
   1-4 (천안역 부동산)
   2-3 (근처 아파트)
   2-4 (근처 부동산)
   1-3-4 (천안역 아파트 급매)
   ```

4. **조합 실행**: "키워드 조합하기" 클릭

5. **결과 복사**: Ctrl+A (전체 선택) → Ctrl+C (복사)

### 2-2. 엑셀 정리 및 1,000개 단위 분리
1. **엑셀 붙여넣기**: 빈 셀에 Ctrl+V

2. **1,000개 단위 분리 방법:**
   ```
   1) 1번째 키워드부터 1,000번째까지 카운트
   2) 1,001번째 키워드 클릭
   3) Ctrl + Shift + ↓ (맨 아래까지 선택)
   4) Ctrl + X (잘라내기)
   5) 옆 빈 셀로 이동
   6) Ctrl + V (붙여넣기)
   7) 위 과정 반복
   ```

3. **결과**: 5-6개의 열로 키워드 분리 완료

## 🏗️ STEP 3: 네이버 파워링크 캠페인 세팅
### 3-1. 캠페인 생성
1. **로그인**: searchad.naver.com 접속
2. **광고 플랫폼** 클릭
3. **새 캠페인** 생성
   - **캠페인 유형**: 파워링크
   - **캠페인 이름**: `iusell 천안아산 부동산 파워링크`
   - **하루 예산**: `5,000원` (월 15만원 수준)
     - 예산 조정 가능 (3,000원~10,000원)
4. **저장 후 닫기**

### 3-2. 확장소재 설정 (전화번호, 네이버예약, 플레이스)
#### 📞 전화번호 확장소재
1. 캠페인 내 **확장소재** 클릭
2. **새 확장소재** → **전화번호** 선택
3. **새로 만들기** 클릭
   - **채널 이름**: `iusell 천안아산부동산`
   - **전화번호**: 0507번호 또는 대표번호 입력
4. **확인** → **추가** → **저장 후 닫기**

#### 📍 플레이스 정보 확장소재
1. **확장소재** → **새 확장소재** → **플레이스** 선택
2. **채널 추가** 클릭
   - **채널 이름**: `iusell 천안아산부동산`
   - **동의 후 인증하기** 클릭
   - **업체명** 선택 (네이버 플레이스 등록 업체)
3. **전달할 서류**: 아니오
4. **저장 후 닫기**

> [!tip] 네이버 예약
> 부동산은 네이버 예약이 일반적으로 없으므로 생략 가능. 상담 예약 시스템이 있다면 설정.

### 4-2. URL 연결 (플레이스 ID 방식)
#### 방법 1: 웹사이트 URL 사용
```
https://iusell.cheonan-asan.com
```

## ✍️ STEP 5: 광고 소재(문구) 작성

### 5-1. 새 소재 생성

1. 광고 그룹 → **소재** 클릭
2. **새 소재** → **단일형 소재** 선택

> [!warning] 반응형 vs 단일형
> - **반응형**: AI가 자동 생성 (부동산과 맞지 않을 수 있음)
> - **단일형**: 직접 작성 (통제 가능, 권장)

### 5-2. 제목 작성 (15자 이내)

**템플릿:**
```
[키워드 삽입] + [브랜드/특징]
```

**예시:**
```
{KeyWord:천안아산부동산} iusell
{KeyWord:천안아파트} 실거래가조회
{KeyWord:아산전세} 급매물모음
```

> [!tip] 키워드 삽입 기능
> `{KeyWord:기본값}` 형식으로 사용자 검색어가 자동 삽입됨

### 5-3. 설명 작성 (45자 이내)
**템플릿:**
```
[키워드 삽입] + [핵심 가치 제안] + [행동 유도]
```

**예시:**
```
{KeyWord:천안아산} 실시간 매물정보! 복비 절감, 빠른 거래 도와드립니다
{KeyWord:아파트} 시세조회부터 계약까지 원스톱! 지금 상담하세요
급매물 다수 보유! 천안아산 전문 중개사가 직접 상담해드립니다
```

### 5-4. 금지 단어 주의
❌ **사용 불가 단어:**
```
1등, 최고, 최대, 최저, 전문, 으뜸, 넘버원, 보장
```

✅ **대체 표현:**
```
다년간 경험, 풍부한 매물, 신속한 거래, 정확한 시세
```

### 5-5. 소재 저장
- **점수 요청**: 아니오
- **저장 후 닫기**

> [!tip] 소재 다양화
> 최소 3-5개의 소재를 생성하여 A/B 테스트 진행 권장

## 💳 STEP 8: 비즈머니 충전
### 8-1. 충전 방법

1. 검색광고 화면 우측 상단 **비즈머니 충전하기**
2. 충전 금액 선택:
   - **최소 충전**: 5만원 (1개월 운영)
   - **권장 충전**: 10-20만원 (2-4개월 운영)
3. 결제 수단 선택 (신용카드, 계좌이체)
4. 충전 완료

### 8-2. 계정 활성화
> [!warning] 중요
> 비즈머니를 충전해야 네이버에서 광고를 검수하고 노출 시작!

- 충전 후 1-2일 이내 검수 완료
- 검수 통과 시 광고 자동 노출 시작

### 💡 저비용 고효과 전략
1. **랜드마크 키워드 집중 공략**
   - 예: `천안역 부동산`, `탕정신도시 아파트`
   - 월 검색량 높지만 경쟁 낮음
   - 기본 입찰가(80원)로도 상위 노출 가능

2. **아파트 단지명 키워드**
   - 예: `두정푸르지오 전세`, `탕정아이파크 매매`
   - 구체적 니즈 → 높은 전환율
   - 경쟁 거의 없음

3. **위치 기반 키워드**
   - `근처 부동산`, `주변 아파트`
   - 모바일 사용자 타겟팅
   - 즉시 방문 가능성 높음

### ⚡ 시간대별 입찰가 조정 (고급)
```
평일 오전 9시-12시: 입찰가 +20% (출근 시간 검색 많음)
평일 오후 6시-9시: 입찰가 +30% (퇴근 후 검색 많음)
주말 오후 2시-6시: 입찰가 +40% (현장 방문 전 검색)
```

### 🎯 품질지수 올리기
**품질지수 = 클릭률 + 관련성 + 랜딩페이지 품질**
1. **클릭률 개선**:
   - 매력적인 제목/설명
   - 이미지 활용
   - 확장소재 모두 연결

2. **관련성 강화**:
   - 키워드와 소재 일치도 높이기
   - 랜딩페이지 키워드 포함

3. **랜딩페이지 최적화**:
   - 로딩 속도 개선
   - 모바일 최적화
   - 명확한 행동 유도(CTA)
