## 🎯 목표 및 의도

### 이 결과물의 목적
- **즉시 실행 가능**: 광고 지식 없어도 따라할 수 있는 단계별 가이드
- **부동산 특화**: iusell.cheonan-asan.com에 최적화된 실전 전략
- **효율 극대화**: 최소 비용으로 최대 성과를 내는 세팅 방법

### 독자가 얻어갈 것
1. 30분 안에 첫 광고 캠페인 생성 가능
2. 부동산에 효과적인 키워드 100개 리스트
3. 매일 10분으로 할 수 있는 광고 관리 루틴
4. 데이터 기반 최적화 방법

### 성공 지표
- **정량적**: 첫 1주일 내 10건 이상의 전화 문의, 클릭당 비용 1,000원 이하
- **정성적**: 광고 운영에 대한 자신감, 데이터 해석 능력 향상

## ⚡ Quick Start: 30분 만에 첫 광고 시작하기

### STEP 0: 사전 준비 (5분)
- [x] 네이버 광고 계정 생성: [searchad.naver.com](https://searchad.naver.com)  [completion:: 2025-11-03]
- [x] 사업자 등록 및 결제 수단 등록  [completion:: 2025-11-03]
- [x] 초기 충전금: 최소 10만원 (테스트용)  [completion:: 2025-11-03]

### STEP 1: 캠페인 생성 (5분)
```
1. "검색광고" → "파워링크" 선택
2. 캠페인 이름: "천안아산 부동산_모바일"
3. 하루 예산: 30,000원
4. 노출 기간: 상시
5. 프리미엄 로그 분석: ✅ 활성화 (필수!)
```

### STEP 2: 광고그룹 생성 (5분)
```
1. 광고그룹 이름: "천안_아파트매매_모바일"
2. 노출 기기: 모바일만 선택
3. 노출 지역: 천안시, 아산시
4. 노출 시간: 09:00 ~ 22:00 (필요시 조정)
```

### STEP 3: 키워드 등록 (10분)
```
1. 핵심 키워드 10개부터 시작 (아래 리스트 참고)
2. 입찰가: 네이버 추천 입찰가의 70% (예: 추천 1,000원 → 700원 입찰)
3. 매칭 옵션: "구매의도 높은 고객" 또는 "광범위" 선택
```

```
- A등급: 전환 1건 + CPA 10만원 이하 → 입찰가 유지
- B등급: 클릭 10회 이상, 전환 0건 → 입찰가 -30%
- C등급: 클릭 5회 미만, 전환 0건 → 입찰가 -50% or 삭제
```

- [ ] S등급 키워드 → 입찰가 상향 (예: 700원 → 840원)
- [ ] A등급 키워드 → 입찰가 유지
- [ ] B등급 키워드 → 입찰가 하향 (예: 700원 → 490원)
- [ ] C등급 키워드 → 일시정지 or 삭제

#### STEP 3: 신규 키워드 발굴 (10분)
- [ ] 성과 좋은 키워드의 유사 키워드 5개 추가
- [ ] 네이버 키워드 도구에서 추천 키워드 확인
- [ ] 경쟁사 광고 검색해서 벤치마킹

#### STEP 4: 광고 소재 A/B 테스트 (5분)
- [ ] 클릭률 낮은 광고그룹의 문구 변경
- [ ] 2가지 버전으로 테스트 (기존 vs 신규)
- [ ] 1주일 후 성과 비교

## ⚠️ 자주 하는 실수 & 해결법

### 실수 1: 무작정 높은 입찰가 설정
**문제**: "1위 해야 한다"는 생각에 추천 입찰가 200% 설정
**결과**: 광고비만 빠르게 소진, 효율 떨어짐
**해결**: 추천 입찰가의 70%부터 시작, 데이터 보고 조정

### 실수 2: PC/모바일 구분 안 함
**문제**: 하나의 광고그룹에 PC와 모바일 함께 운영
**결과**: 디바이스별 최적화 불가, 비효율 발생
**해결**: 반드시 PC/모바일 분리 광고그룹 생성

### 실수 3: 키워드 10개만 등록
**문제**: 핵심 키워드 10개만 등록하고 방치
**결과**: 경쟁 치열한 키워드만 소진, 기회 상실
**해결**: 최소 50-100개 키워드로 롱테일 공략

### 실수 4: 전환추적 안 함
**문제**: 프리미엄 로그 분석 미설정
**결과**: 어떤 키워드가 실제 성과 내는지 모름
**해결**: 첫날부터 전환추적 필수 설정

### 실수 5: 방치형 운영
**문제**: 한 번 설정하고 한 달 방치
**결과**: 비효율 키워드가 광고비 잡아먹음
**해결**: 주 1회 30분 최적화 필수

## 🎓 고급 팁 & 트릭

### Tip 1: 시간대별 입찰 조정
```
전환 높은 시간대:
- 평일 10-12시: 입찰가 +20%
- 평일 19-21시: 입찰가 +20%
- 주말 전체: 입찰가 +10%

전환 낮은 시간대:
- 새벽 0-6시: 입찰가 -50% or 노출 중단
```

### Tip 2: 날씨 연동 전략
```
비 오는 날: "실내 보러 오세요" 문구 + 입찰가 -10%
날씨 좋은 날: "매물 방문하기 좋은 날씨" + 입찰가 +10%
```

### Tip 3: 계절별 키워드
```
봄 (3-5월): 이사 시즌 → "빠른 입주" 강조
여름 (6-8월): 비수기 → 할인 이벤트 강조
가을 (9-11월): 재계약 시즌 → "전세 대란" 키워드
겨울 (12-2월): 연말 절세 → "증여", "절세" 키워드
```

### Tip 4: 경쟁사 키워드 역공
```
경쟁사 광고 검색 → 어떤 키워드 쓰는지 분석
→ 같은 키워드에 더 매력적인 문구로 대응
→ 경쟁사가 놓친 롱테일 키워드 발굴
```

### Tip 5: 품질지수 해킹
```
품질지수 향상 = 광고비 절감

방법 1: 키워드와 광고 문구 일치도 높이기
- 키워드 "천안아파트" → 제목에 "천안아파트" 반드시 포함

방법 2: 랜딩페이지에 키워드 포함
- 키워드 "천안아파트" → 랜딩페이지 H1 태그에 "천안아파트" 배치

방법 3: 클릭률 높이기
- 확장소재 풀 활용 → 노출 면적 2배 → 클릭률 상승 → 품질지수 상승
```
