# 📤 Series A EP.01 유튜브 업로드 최적화 가이드

## 🎯 영상 제목 (Title)

### 메인 제목 (선택 옵션)

#### 옵션 1: 클릭 유도형 (추천 ⭐)
```
아산 유일! 단지 내 수영장 있는 아파트 실제로 가봤습니다 | 우방아이유쉘
```
**길이:** 44자 (모바일 완전 노출)
**전략:** 호기심 유발 ("유일", "실제로") + 브랜드명

#### 옵션 2: 팩트 강조형
```
아산배방 수영장 아파트? 1억 저렴한데 즉시입주 가능! (시설 투어)
```
**길이:** 38자
**전략:** 가격 메리트 + 즉시 입주 강조

#### 옵션 3: 질문형 (호기심)
```
집에서 30초 거리에 수영장이? 아산 유일 우방아이유쉘 리얼 후기
```
**길이:** 36자
**전략:** 질문으로 호기심 자극 + 리얼 후기

#### 옵션 4: 타겟 명확형
```
아이 키우기 딱 좋은 아파트! 수영장+초품아 아산배방 우방아이유쉘
```
**길이:** 37자
**전략:** 타겟(유자녀 가족) 명확히 + 핵심 시설

### 제목 최적화 원칙
✅ **해야 할 것:**
- 핵심 키워드 앞쪽 배치 ("아산", "수영장", "아파트")
- 숫자 활용 ("1억 저렴", "30초")
- 감정 유발 단어 ("유일", "실제로", "딱 좋은")
- 50자 이내 (모바일 완전 노출)
- 브랜드명 포함 (검색 최적화)

❌ **하지 말아야 할 것:**
- 과장된 표현 ("대박!", "미친!")
- 클릭베이트 ("절대 믿을 수 없는", "충격적인")
- 이모티콘 과다 사용
- 허위 정보
- 60자 초과 (모바일 잘림)
