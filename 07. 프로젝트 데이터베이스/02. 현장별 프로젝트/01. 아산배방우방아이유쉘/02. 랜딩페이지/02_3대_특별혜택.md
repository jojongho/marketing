```
┌─────────────────────────────────────────┐
│  일반 중개사무소  vs  단지 내 공인중개사  │

│  절약금액:   780만원                      │

```

### 상세 설명
```
왜 중개수수료가 없나요?

✅ 단지 내 공인중개사 직접 운영
✅ 시행사 공식 제휴 업체
✅ 분양전환 특별 프로모션
✅ 중간 마진 없는 직거래

일반적으로 2.6억 기준
매수자 0.4% = 104만원
매도자 0.4% = 104만원
임대 시 최대 0.5% = 130만원

→ 모두 0원으로 절약!
```

```

   - 제휴 가구 업체 할인 (10-20%)
   - 가전제품 업체 특가 연결
   - 인테리어 업체 소개

🎨 인테리어 지원
   - 필수 보수 공사 지원

   - 중도금 대출 연결
   - 전세자금 대출 상담
```

## Section 7: 자주 묻는 질문

### FAQ (확장형)

#### Q1. 정말 조건 없이 모두 받을 수 있나요?
```
A. 네, 맞습니다.
계약하시는 모든 분께 동일하게 제공됩니다.
평수, 가격, 대출 여부와 무관하게
모든 계약자가 혜택을 받으실 수 있습니다.
```

#### Q2. 중개수수료가 정말 0원인가요?
```
A. 네, 전액 무료입니다.
단지 내 공인중개사를 통해 계약하시면
매수, 매도, 임대 모두 중개수수료가 없습니다.
일반적으로 2.6억 기준 780만원이 발생하지만
우리는 0원입니다.
```

#### Q3. 계약축하금은 언제 받나요?
```
A. 계약 즉시 지급됩니다.
계약서 작성 완료 후 바로 계좌 이체로
1,000만원을 지급해 드립니다.
별도의 대기 기간이나 조건이 없습니다.
```

#### Q4. 입주지원금은 어떻게 받나요?
```
A. 상담 시 맞춤 안내해 드립니다.
이사비용, 가구·가전 할인, 인테리어 등
필요하신 항목에 맞춰 지원해 드립니다.
상담 시 자세히 안내받으실 수 있습니다.
```

#### Q5. 혜택에 세금이 부과되나요?
```
A. 아니요, 세금 없습니다.
계약축하금과 입주지원금은 프로모션 혜택으로
증여세 등 별도 세금이 발생하지 않습니다.
```

#### Q6. 언제까지 혜택을 받을 수 있나요?
```
A. 조기 마감될 수 있습니다.
분양전환 특별 프로모션으로 기간 한정입니다.
잔여 세대 소진 시 혜택이 종료될 수 있으니
빠른 상담을 권장드립니다.
```
