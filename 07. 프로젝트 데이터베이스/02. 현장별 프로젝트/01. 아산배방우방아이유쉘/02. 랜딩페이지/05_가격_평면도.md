## 🏠 평형별 상세 평면도

### 25평형 (61㎡) - 신혼부부·소가족 추천

**61A 타입 (600세대)**
```
📐 구조: 3룸 (방 2개 + 거실)
🛏️ 방: 안방(침대+화장대), 침실1(책상+침대)
🍳 주방: ㄱ자 주방 + 팬트리
🛁 욕실: 2개 (안방 욕실 + 공용 욕실)
🪟 발코니: 거실 확장형 + 안방 확장형

[평면도 이미지]
현관 → 거실(발코니 확장) → 주방
        ↓
안방(발코니 확장) + 안방 욕실
침실1 + 공용 욕실
```

**61B 타입 (500세대)**
```
📐 구조: 3룸 + 드레스룸
🛏️ 방: 안방(드레스룸), 침실1, 침실2
🍳 주방: 一자 주방 + 다용도실
🛁 욕실: 2개
🪟 발코니: 3면 확장 (거실+안방+침실1)

[평면도 이미지]
현관 → 거실(넓은 발코니) → 주방
        ↓
안방(드레스룸+욕실) + 침실1 + 침실2
```

**61C 타입 (300세대)**
```
📐 구조: 4룸 (방 3개 + 거실)
🛏️ 방: 안방, 침실1, 침실2
🍳 주방: ㄱ자 주방 + 팬트리
🛁 욕실: 2개
🪟 발코니: 4면 확장

[평면도 이미지]
최대 공간 활용형
방 3개로 아이 2명 가족 최적
```

---

### 31평형 (78㎡) - 가족·실거주 추천

**78A 타입 (300세대)**
```
📐 구조: 4룸 (방 3개 + 거실)
🛏️ 방: 안방(드레스룸), 침실1, 침실2
🍳 주방: ㄱ자 주방 + 팬트리 + 다용도실
🛁 욕실: 2개 (안방 욕실 욕조)
🪟 발코니: 5면 확장 (거실+안방+침실+주방)

[평면도 이미지]
현관 → 넓은 거실(발코니 확장) → 주방(다용도실)
        ↓
안방(드레스룸+욕조 욕실) + 침실1 + 침실2
파우더룸 + 공용 욕실
```

**78B 타입 (86세대)**
```
📐 구조: 4룸 + 서재
🛏️ 방: 안방(대형 드레스룸), 침실1, 침실2, 서재
🍳 주방: 一자 주방 + 식당 공간 분리
🛁 욕실: 3개 (안방+공용+파우더룸)
🪟 발코니: 전면 확장

[평면도 이미지]
프리미엄 타입
서재 별도 공간 (재택근무 최적)
안방 드레스룸 대형
```

├─ 금리 우대: 0.5-1%
