
## 🎯 페이지별 목적 및 전환 전략

### 1. 메인 홈페이지 (/)
**목적**: 첫 인상 + 즉각적 관심 유도
**핵심 메시지**: "중개수수료 없음 + 1,000만원 축하금"
**CTA**: "상담 신청하기", "전화 상담"

### 2. 단지 소개 (/about)
**목적**: 신뢰도 구축
**핵심 메시지**: 우방건설 브랜드, 1,786세대 대단지
**CTA**: "특별혜택 보기"

### 3. 3대 특별혜택 (/benefits)
**목적**: 차별화 요소 강조
**핵심 메시지**:
- 중개수수료 없음 (780만원 절약)
- 계약축하금 1,000만원
- 입주지원금 지원
**CTA**: "지금 상담 신청"

### 4. 특화시설 (/facilities)
**목적**: 프리미엄 이미지
**핵심 메시지**:
- 아산 최초 단지 내 수영장
- 초품아 (초등학교 품은 아파트)
**CTA**: "시설 둘러보기", "상담하기"

### 5. 입지 정보 (/location)
**목적**: 생활 편의성 강조
**핵심 메시지**:
- 삼성디스플레이 도보 5분
- 배방역 도보 5분
- 생활편의시설 도보권
**CTA**: "주변 시설 보기"

### 6. 가격 & 평면도 (/price)
**목적**: 구매 결정 지원
**핵심 메시지**: 투명한 가격 공개
**CTA**: "상세 가격표 받기", "상담 신청"

### 7-1. 신혼부부 페이지 (/for/newlyweds)
**타겟**: 25-35세 신혼부부
**핵심 메시지**: 25평 2억대, 첫집 마련
**CTA**: "신혼부부 혜택 확인"

### 7-2. 직장인 페이지 (/for/workers)
**타겟**: 25-45세 삼성디스플레이 직장인
**핵심 메시지**: 워라밸, 도보출근
**CTA**: "출퇴근 시간 계산해보기"

### 7-3. 가족 페이지 (/for/family)
**타겟**: 30-45세 초등자녀 가족
**핵심 메시지**: 초품아, 안전한 등하교
**CTA**: "교육환경 상세보기"

### 7-4. 투자자 페이지 (/for/investors)
**타겟**: 35-55세 부동산 투자자
**핵심 메시지**: 즉시입주, 임대수익
**CTA**: "투자 가치 분석 보기"

### 8. 상담신청 (/contact)
**목적**: 전환 완성
**폼 구성**: 이름, 전화번호, 희망 평수, 방문 희망일
**CTA**: "1분 만에 상담 신청"

### 9. 분양전환 FAQ (/faq)
**목적**: 궁금증 해소
**핵심 내용**:
- 분양전환이란?
- 입주 가능 시기는?
- 대출은 가능한가요?
**CTA**: "더 궁금하면 전화주세요"
