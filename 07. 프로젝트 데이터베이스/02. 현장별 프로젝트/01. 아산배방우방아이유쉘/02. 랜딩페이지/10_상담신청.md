## Section 3: 상담 프로세스

### 타이틀
```
🔄 상담부터 계약까지
```

### 프로세스 플로우
```
STEP 1: 상담 신청
└→ 📝 폼 작성 또는 전화/카톡

STEP 2: 전문 상담
└→ 💬 전문 상담사 배정
   └→ 평수, 가격, 혜택 안내
   └→ 대출, 입주 일정 상담

STEP 3: 현장 방문 (선택)
└→ 🏢 단지 현장 안내
   └→ 실제 세대 확인
   └→ 시설 둘러보기

STEP 4: 계약 진행
└→ 📄 계약서 작성
   └→ 중개수수료 0원 확인
   └→ 계약축하금 1,000만원 지급

STEP 5: 입주 준비
└→ 🏠 입주지원금 지원
   └→ 이사, 인테리어 연결
   └→ 대출 진행

STEP 6: 입주 완료
└→ ✅ 즉시입주 가능
   └→ 새로운 시작!
```

### 소요 시간
```
상담: 20-30분
현장 방문: 30-60분
계약: 30-60분

총 소요 시간: 1-2시간
(당일 계약 가능)
```

## Section 4: 상담 FAQ

### 자주 묻는 질문

#### Q1. 상담만 받아도 되나요?
```
A. 네, 물론입니다!
상담만 받으시는 것도 환영합니다.
계약 압박 전혀 없으니 편하게 상담받으세요.
궁금한 점 모두 해결하고 결정하시면 됩니다.
```

#### Q2. 상담 비용이 있나요?
```
A. 아니요, 완전 무료입니다.
상담, 현장 방문, 계약 상담 모두 무료입니다.
어떤 비용도 발생하지 않으니 안심하세요.
```

#### Q3. 당일 계약도 가능한가요?
```
A. 네, 가능합니다.
상담 후 마음에 드시면 당일 계약 진행하실 수 있습니다.
필요 서류만 준비하시면 즉시 진행 가능합니다.

필요 서류:
- 신분증
- 인감증명서 (또는 본인서명사실확인서)
- 계약금 (체크 또는 현금)
```

#### Q4. 주말에도 상담 가능한가요?
```
A. 네, 주말 상담 가능합니다.
평일 09:00-18:00
주말 10:00-17:00
모두 상담 가능합니다.
주말이 더 편하시면 주말 상담 신청하세요.
```

#### Q5. 방문 상담 시 주차는 어떻게 하나요?
```
A. 단지 내 주차장 이용 가능합니다.
방문 시 단지 내 주차장에 주차하시면 됩니다.
주차 공간 충분하니 걱정하지 마세요.
```

#### Q6. 대출 상담도 받을 수 있나요?
```
A. 네, 대출 상담 가능합니다.
중도금 대출, 전세자금 대출 등
필요하신 대출 상담 모두 제공합니다.
제휴 은행 연결로 최저 금리 안내해 드립니다.
```
