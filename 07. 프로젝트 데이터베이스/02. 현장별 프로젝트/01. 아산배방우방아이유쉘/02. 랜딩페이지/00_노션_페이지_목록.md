## 🚀 Notion Bullet Builder 구현 가이드

### 페이지 속성 설정 방법

각 페이지 생성 시 다음 속성을 추가:

1. **Path**: URL 경로 (예: `/`, `/benefits`, `/for/newlyweds`)
2. **Page Type**: Static Page (모든 페이지)
3. **Publish**: ✅ 체크 (활성화)
4. **Meta Title**: SEO 최적화 제목 (50-60자)
5. **Meta Description**: SEO 최적화 설명 (120-160자)
6. **Meta Keywords**: 핵심 키워드 5-8개

### 페이지 생성 순서

**1단계 (즉시 구현)**
1. 메인 홈페이지 (`/`)
2. 3대 특별혜택 (`/benefits`)
3. 상담신청 (`/contact`)

**2단계 (3일 내)**
4. 특화시설 (`/facilities`)
5. 입지정보 (`/location`)
6. 가격 & 평면도 (`/price`)

**3단계 (1주일 내)**
