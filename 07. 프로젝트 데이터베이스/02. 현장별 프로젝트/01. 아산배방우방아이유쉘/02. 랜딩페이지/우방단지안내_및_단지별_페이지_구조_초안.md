# 🏗️ 우방단지안내 및 단지별 페이지 구조 초안

## 1. 📂 메뉴 및 사이트 구조 (Sitemap)

기존 메뉴 구조에 **'우방단지안내'** 메뉴를 추가하고, 하위에 **'1단지'**와 **'2단지'**를 배치합니다.

```
/ (홈)
...
├─ /complex-guide (우방단지안내) [NEW]
│  ├─ /complex-guide/complex1 (1단지) [NEW]
│  └─ /complex-guide/complex2 (2단지) [NEW]
...
```

---

## 2. 📄 페이지별 상세 구조

### 2-1. 우방단지안내 (메인/공통) (`/complex-guide`)
**목적**: 1단지와 2단지의 공통적인 특장점과 브랜드 가치를 간단명료하게 전달하고, 각 단지 상세 페이지로 유도합니다.

**구성 요소**:
1.  **헤더 (Header)**: 공통 네비게이션
2.  **인트로 (Intro)**:
    *   메인 카피: "아산 배방의 자부심, 우방 아이유쉘" (예시)
    *   브랜드 소개: 우방건설의 브랜드 가치 및 신뢰도
    *   대단지 프리미엄: 1, 2단지 합쳐서 누리는 대단지 라이프 강조
3.  **단지 선택 (Navigation/Links)**:
    *   **[1단지 보러가기]** 버튼/카드
    *   **[2단지 보러가기]** 버튼/카드
4.  **공통 프리미엄 요약**:
    *   공통적으로 적용되는 혜택이나 입지적 장점 간단 나열 (3대 특별혜택 등 리마인드)
5.  **푸터 (Footer)**

---

### 2-2. 1단지 상세 페이지 (`/complex-guide/complex1`)
**형식**: 원페이지 (One-page) 스크롤 방식
**목적**: 1단지만의 구체적인 정보를 제공하여 입주 의향을 높임.

**섹션 구성**:

1.  **섹션 1: 입지 (Location)**
    *   **지도/약도**: 1단지의 정확한 위치 표시
    *   **주요 접근성**: 배방역, 삼성디스플레이, 학교 등과의 거리 강조
    *   **주변 인프라**: 1단지 주변 편의시설 아이콘/텍스트

2.  **섹션 2: 프리미엄 (Special Features)**
    *   **1단지 특장점**: 1단지만이 가진 특별한 장점 (예: 조망권, 특정 커뮤니티 접근성 등)
    *   **생활 혜택**: 삶의 질을 높여주는 요소들

3.  **섹션 3: 단지배치도 (Complex Layout)**
    *   **배치도 이미지**: 고화질 단지 배치도
    *   **동선 안내**: 출입구, 주차장, 산책로 등 표시
    *   **방향 표시**: 남향 위주 배치 강조

4.  **섹션 4: 동호수배치도 (Unit Layout)**
    *   **동별 층수/호수표**: 각 동의 층수와 라인별 타입 표시
    *   **필로티/특화층**: 필로티 유무 및 최상층 등 특이사항 표시

5.  **섹션 5: 평면정보 (Floor Plan)**
    *   **타입별 평면도**: (예: 59㎡, 84㎡ 등) 탭(Tab) 또는 슬라이드 형식으로 구성
    *   **아이소매트릭(ISO)**: 입체 평면도 (있을 경우)
    *   **상세 스펙**: 전용면적, 공용면적, 서비스면적 등 표기

6.  **하단 CTA (Call to Action)**
    *   "1단지 잔여세대 문의하기"
    *   상담신청 폼 또는 전화 연결 버튼

---

### 2-3. 2단지 상세 페이지 (`/complex-guide/complex2`)
**형식**: 원페이지 (One-page) 스크롤 방식
**목적**: 2단지만의 구체적인 정보를 제공.

**섹션 구성**: (1단지와 동일한 구조, 내용은 2단지 맞춤)

1.  **섹션 1: 입지 (Location)**
    *   2단지 위치 기준 주변 정보 및 접근성

2.  **섹션 2: 프리미엄 (Special Features)**
    *   2단지 특화 장점 (예: 수영장 인접, 학교 인접 등 2단지만의 강점)

3.  **섹션 3: 단지배치도 (Complex Layout)**
    *   2단지 배치도 이미지

4.  **섹션 4: 동호수배치도 (Unit Layout)**
    *   2단지 동호수표

5.  **섹션 5: 평면정보 (Floor Plan)**
    *   2단지 타입별 평면도 및 정보

6.  **하단 CTA**
    *   "2단지 로얄동호수 문의하기"
    *   상담신청 폼 또는 전화 연결 버튼
