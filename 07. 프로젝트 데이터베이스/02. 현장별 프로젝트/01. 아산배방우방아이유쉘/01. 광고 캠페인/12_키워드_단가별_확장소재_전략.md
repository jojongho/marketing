# 🎯 키워드 단가별 확장소재 차별화 전략

## 📋 전략 개요

네이버 파워링크 키워드를 **단가(CPC)에 따라 2가지 전략**으로 분리하여 운영:
1. **고단가 키워드 (150원↑)**: 직접 전환 목표 → 랜딩페이지 직행
2. **저단가 키워드 (70-100원)**: 브랜드 노출 목표 → 네이버 플레이스/블로그 활용 (에펠탑 효과)

### 핵심 인사이트
- **부정클릭 리스크 관리**: 플레이스는 부정클릭 방지 프로그램 설치 불가 → 저단가 키워드로 리스크 완화
- **확장소재 차별화**: 플레이스 연동 시 사무실 이미지 노출 가능 (전화번호 포함 허용)
- **플레이스 최적화**: 저단가 키워드로 플레이스 유입 증가 → 플레이스 순위 상승 → 자연 유입 증대

## 🛡️ 부정클릭 방어 전략

### 플레이스의 한계와 저단가 전략의 필연성

```
⚠️ 플레이스는 부정클릭 방지 프로그램 설치 불가
   - 일반 홈페이지: 헤더에 부정클릭 방지 스크립트 설치 가능
   - 네이버 플레이스: 외부 스크립트 설치 불가능
   - → 부정클릭에 무방비 상태

💡 따라서 플레이스는 저단가 전략만 가능
   - 클릭당 70-100원 → 부정클릭 당해도 타격 적음
   - 에펠탑 효과 (다중 노출) 목적
   - 클릭 많이 해도 예산 부담 적음
   - 부정클릭 방지 프로그램이 딱히 필요 없음

✅ 저단가라서 가능한 플레이스 전략
   - 일 예산 20,000원 / 클릭당 80원 = 250클릭
   - 부정클릭 50% 발생해도 일 10,000원 손실
   - 고단가였다면 일 50,000원 이상 손실 가능성
   - 저단가이기 때문에 리스크 감수 가능
```

### 고단가 키워드 부정클릭 방어 (랜딩페이지 직행)

```
✅ 헤더 스크립트 설치 (필수)
   - 부정클릭 방지 프로그램 스크립트
   - 랜딩페이지 <head> 태그 내 설치
   - 의심 IP 자동 차단
   - 반복 클릭 패턴 감지

✅ 일일 예산 제한
   - 고단가 캠페인: 일 30,000원
   - 그룹별 상한 설정
   - 예산 소진 시 자동 정지

✅ 수동 모니터링
   - IP 차단 (네이버 광고 관리)
   - 동일 IP 반복 클릭 감지
   - 시간대별 클릭 패턴 분석
   - 비정상 패턴 감지 시 일시정지

✅ 전환 추적 연동
   - 클릭만 많고 전환 없는 IP 식별
   - 실제 전환 발생 IP만 유지
   - ROI 기반 필터링
```

### 전략별 부정클릭 리스크 비교

| 구분 | 고단가 (랜딩페이지) | 저단가 (플레이스) |
|------|---------------------|-------------------|
| CPC | 150-250원 | 70-100원 |
| 방지 프로그램 | ✅ 설치 가능 | ❌ 설치 불가 |
| 부정클릭 50% 발생 시 | 일 15,000원 손실 | 일 10,000원 손실 |
| 리스크 | 높음 (방지 필수) | 낮음 (저단가로 완화) |
| 전략 | 방어 + 전환 극대화 | 리스크 감수 + 대량 노출 |
