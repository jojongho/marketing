# 🥕 당근마켓 전문가 모드 실행 단계별 가이드

## 📋 사전 준비 체크리스트

### 1. 계정 준비
- [ ] 당근마켓 일반 계정 보유 (없으면 가입)
- [ ] 사업자 등록증 준비
- [ ] 업체 정보 정리 (상호명, 주소, 전화번호)

### 2. 콘텐츠 준비
- [ ] 프로필 사진 (1:1 비율, 최소 1장)
- [ ] 단지 사진 (최소 5장, 1:1 비율 권장)
- [ ] 소개글 작성 완료
- [ ] 소식 3~5개 작성 완료

### 3. 광고 소재 준비
- [ ] 광고 이미지 4~5개 (1:1 비율)
- [ ] 광고 제목 10~15개 (소재별 2~3개)
- [ ] 네이버 플레이스 주소 확인

### 4. 예산 준비
- [ ] 광고 예산 확보 (초기 10만원 권장)
- [ ] 결제 수단 준비

## 🚀 1단계: 당근 비즈니스 계정 생성

### Step 1-1: 당근 비즈니스 사이트 접속
1. 구글에서 **"당근 비즈니스"** 검색
2. 또는 직접 접속: `https://business.daangn.com`
3. **시작하기** 버튼 클릭

### Step 1-2: 로그인
- **네이버**, **카카오**, **구글** 중 선택
- QR코드로 핸드폰 로그인 가능 (PC에서)

### Step 1-3: 비즈 프로필 생성
1. **새 비즈 프로필** 클릭
2. **업체명 입력**
   - 예: "배방우방아이유쉘" 또는 "아산배방우방아이유쉘"
   - 실제 상호명과 일치하게 입력
3. **업종 선택**
   - 부동산 중개업 또는 부동산 개발업
   - 사업자 등록증의 종목과 일치
   - ⚠️ 생성 후 변경 불가
4. **주소 입력**
   - 충남 아산시 배방읍 공수리 (정확한 주소)
   - 고객이 방문할 수 있는 실제 주소
5. **다음** 클릭

### Step 1-4: 프로필 정보 입력
1. **프로필 사진 업로드**
   - 단지 대표 이미지 (1:1 비율 권장)
2. **사진 추가** (최소 5장)
   - 단지 외관, 수영장, 커뮤니티 시설 등
3. **소개글 입력**
   - 템플릿 참고하여 작성
4. **전화번호 입력**
   - 상담 전화번호
5. **업체 정보 입력**
   - 홈페이지 링크 (네이버 플레이스)
   - 단골 혜택 설정
6. **저장** 클릭

✅ **비즈 프로필 생성 완료!**

---

## 🚀 2단계: 광고 계정 생성 (전문가 모드)

### Step 2-1: 광고 계정 만들기
1. 우측 **광고 계정** 메뉴 클릭
2. **새 광고 계정** 클릭
3. **전문가 모드 시작하기** 선택 ⭐ (간편 모드 X)

### Step 2-2: 광고 계정 정보 입력
1. **광고 계정 이름** 입력
   - 예: "배방우방아이유쉘_광고계정"
   - 편한 이름으로 설정 가능
2. **사업자 등록번호** 입력
3. **다음** 클릭

### Step 2-3: 사업자 정보 확인
1. **사업자 정보 가져오기** 클릭
2. 사업자 정보 확인
3. 종사업장 번호 입력 (해당 시)
4. 세금 계산서 발행용 정보 확인
5. **광고 계정 만들기** 클릭

✅ **광고 계정 생성 완료!**

---

## 🚀 3단계: 광고 캐시 충전

### Step 3-1: 캐시 충전
1. 광고 계정 대시보드에서 **광고 캐시** 메뉴 클릭
2. **충전하기** 클릭
3. **10만원** 충전 (초기 권장)
4. 결제 수단 선택 및 결제

### Step 3-2: 자동 충전 설정 (선택)
- 캐시 소진 시 자동 충전 설정 가능
- 초기에는 수동 충전 권장

✅ **캐시 충전 완료!**

## 🚀 4단계: 캠페인 1 생성 (비즈 프로필 알리기)

### Step 4-1: 캠페인 만들기
1. **광고 만들기** 클릭
2. **캠페인 만들기** 클릭
3. 캠페인 이름 확인 (자동 생성, 변경 가능)

### Step 4-2: 캠페인 목표 선택
1. **비즈 프로필 알리기** 선택
2. **알리고자 하는 비즈 프로필** 선택
   - 앞서 생성한 비즈 프로필 선택
3. **다음** 클릭

### Step 4-3: 광고 그룹 설정 (타겟팅)

#### 지역 설정
1. **특정 지역에 게재하기** 선택
2. 기존 설정 지우기
3. 지역 추가:
   - **충청남도 아산시 배방읍**
   - **충청남도 아산시 탕정면**
   - **충청남도 천안시 동남구**
   - **충청남도 천안시 서북구**
4. **완료** 클릭

#### 성별/연령 설정
1. **성별:** 모든 성별
2. **연령:**
   - 주 타겟: 30세 ~ 45세
   - 부 타겟: 40세 ~ 60세 (별도 광고 그룹 권장)
3. **완료** 클릭

#### 모바일 OS
- **안드로이드 + iOS** 모두 선택

#### 관심사 설정
1. **관심사 추가** 클릭
2. 다음 관심사 선택:
   - 부동산 (필수)
   - 주거/인테리어
   - 가족/육아
   - 교육/학원
   - 건강/운동
3. **완료** 클릭

#### 예산 설정
1. **총 예산** 선택 (권장)
2. **총 예산:** 25만원
3. **기간:** 30일
   - 시작일: 오늘
   - 종료일: 30일 후
4. 또는 **1일 예산:** 1만원 (간단한 방식)

#### 입찰 설정
1. **자동 입찰** 선택 (초기)
2. 1주일 후 수동 입찰로 전환 예정

#### 일정 설정
1. **모든 시간에 광고 게재하기** 선택 (초기)
2. 또는 특정 시간대 설정

#### 맞춤 타겟 (선택, 나중에 설정 가능)
- 비즈 프로필 방문자 타겟은 모수 확보 후 설정

3. **다음** 클릭

### Step 4-4: 소재 설정

#### 소재 1: 수영장 강조형
1. **소재 이름:** "수영장 강조형"
2. **소재 유형:** 비즈 프로필 홈 또는 비즈 프로필 소식
3. **비즈 프로필 이미지와 제목 그대로 사용** 체크
   - 또는 **광고 제목 추가:**
     - "아산 최초 단지 내 수영장! 배방 우방아이유쉘"
     - "수영장 있는 아파트, 배방에 있다고?"
4. **소재 추가하기** 클릭

#### 소재 2: 초품아 강조형
1. **소재 이름:** "초품아 강조형"
2. **광고 제목:**
   - "초등학교 품은 아파트, 배방 우방아이유쉘"
   - "공수초 인접 아파트, 배방에 있다고?"
3. **소재 추가하기** 클릭

#### 소재 3: 가격 경쟁력 강조형
1. **소재 이름:** "가격 경쟁력 강조형"
2. **광고 제목:**
   - "주변 시세 대비 1억 저렴! 배방 우방아이유쉘"
   - "2~3억대로 내집 마련, 배방에 있다고?"
3. **소재 추가하기** 클릭

#### 소재 4: 삼성 직주근접 강조형
1. **소재 이름:** "삼성 직주근접 강조형"
2. **광고 제목:**
   - "삼성 나노시티 직주근접! 배방 우방아이유쉘"
   - "삼성 직장인들이 선택하는 아파트?"
3. **소재 추가하기** 클릭

#### 소재 5: 대단지 프리미엄 강조형
1. **소재 이름:** "대단지 프리미엄 강조형"
2. **광고 제목:**
   - "1,786세대 대단지 프리미엄! 배방 우방아이유쉘"
   - "대단지 아파트, 배방에 있다고?"
3. **소재 추가하기** 클릭

### Step 4-5: 최종 확인 및 등록
1. 모든 설정 확인
2. **등록** 클릭
3. 광고 캐시 충전 여부 확인
4. 광고 라이브 대기

✅ **캠페인 1 생성 완료!**

## 🚀 5단계: 캠페인 2 생성 (웹사이트 방문 유도하기)

### Step 5-1: 캠페인 만들기
1. **광고 만들기** 클릭
2. **캠페인 만들기** 클릭

### Step 5-2: 캠페인 목표 선택
1. **웹사이트 방문 유도하기** 선택
2. **추적 코드 연동하지 않습니다** 선택 (오프라인 플레이스)
3. **다음** 클릭

### Step 5-3: 광고 그룹 설정
- 캠페인 1과 동일한 타겟팅 설정
- 또는 약간 다른 타겟팅으로 A/B 테스트

### Step 5-4: 소재 설정

#### 네이버 플레이스 원 주소 확인
1. 네이버 플레이스 접속
2. 업체 페이지 열기
3. **공유** 버튼 클릭
4. 주소 복사
5. **물음표(?) 이전까지의 숫자만** 사용
   - 예: `https://place.naver.com/place/1234567890` ✅
   - 예: `https://place.naver.com/place/1234567890?entry=pll` ❌

#### 소재 생성
1. **소재 이름:** "웹사이트 방문 유도형"
2. **이미지/동영상:** 1:1 비율 이미지 업로드
3. **광고 제목:**
   - "배방 우방아이유쉘 상세 정보 보기"
   - "배방 우방아이유쉘 자세히 알아보기"
4. **브랜드 이름:** "배방우방아이유쉘" (상호명 정확히)
5. **버튼 문구:** "문의하기" 또는 "보러가기"
6. **웹사이트 URL:** 네이버 플레이스 원 주소 입력
7. **소재 추가하기** 클릭

#### 소재 4~5개 생성
- 각 소재마다 다른 이미지와 제목 사용
- A/B 테스트를 위한 다양화

### Step 5-5: 최종 확인 및 등록
1. 모든 설정 확인
2. **등록** 클릭

✅ **캠페인 2 생성 완료!**

   - CPC 추이

2. **전화 문의 수** 추적 (수동)
3. **채팅 문의 수** 확인
4. **네이버 플레이스 유입** 확인

## ⚠️ 문제 해결 가이드

### 광고가 노출되지 않을 때
1. **광고 캐시 확인**
   - 캐시가 부족하면 충전 필요
2. **맞춤 타겟 모수 확인**
   - 모수가 부족하면 수집 기간 연장
3. **심사 상태 확인**
   - 광고 심사 중일 수 있음
4. **타겟팅 범위 확인**
   - 타겟팅이 너무 좁으면 모수 부족

### 클릭률이 낮을 때
1. **소재 확인**
   - 이미지가 매력적인지 확인
   - 제목이 궁금증을 유발하는지 확인
2. **타겟팅 확인**
   - 타겟팅이 정확한지 확인
   - 관심사가 적절한지 확인
3. **소재 교체**
   - 클릭률 낮은 소재 제거
   - 새로운 소재 추가

### 비용이 너무 많이 소진될 때
1. **일 예산 확인**
   - 일 예산이 너무 높으면 조정
2. **입찰가 확인**
   - 수동 입찰로 전환하여 조정
3. **시간대 조정**
   - 효율 낮은 시간대 광고 중단
4. **지역 조정**
   - 효율 낮은 지역 제거
