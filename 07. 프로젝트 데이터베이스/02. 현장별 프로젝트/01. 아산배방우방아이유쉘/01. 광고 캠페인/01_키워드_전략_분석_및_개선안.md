# 🎯 키워드 전략 분석 및 개선안

## 📊 기존 광고 성과 분석

### ❌ 실패한 키워드 전략

#### 문제 1: 부동산 중개업소 키워드
```
사용 키워드:
- 아산 부동산
- 배방읍 공인중개사
- 아산시 공인중개사사무소
- 배방읍 수철리 부동산사무실
```

**문제점:**
- ❌ **의도 불일치**: 중개업소를 찾는 사람 ≠ 분양 아파트 찾는 사람
- ❌ **전환 불가능**: 광고비만 소진, 문의 전환 제로
- ❌ **타겟 오류**: B2B 키워드를 B2C 광고에 사용

**예상 검색자:**
- 매물 내놓으려는 집주인
- 부동산 업체 찾는 사람
- 중개 수수료 비교하는 사람

#### 문제 2: 브랜드명 키워드
```
사용 키워드:
- 배방우방아이유쉘
- 아산배방우방아이유쉘
- 배방우방아이유쉘1단지
```

**문제점:**
- ⚠️ **검색량 극소**: 이미 아는 사람만 검색 (월 10-50회 추정)
- ⚠️ **신규 수요 불가**: 잠재 고객 유입 불가능
- ⚠️ **비효율**: 높은 CPC 대비 낮은 전환

**실제 상황:**
- 이미 관심 있는 극소수만 검색
- 대부분은 현장명 모름
- 광고 노출 자체가 적음

#### 문제 3: 경쟁 현장 키워드
```
사용 키워드:
- 아산탕정자이센트럴시티
- 아산탕정자이퍼스트시티
- 천안휴먼빌퍼스트시티
```

**문제점:**
- ❌ **고객 전환 불가**: 이미 다른 아파트에 관심 집중
- ❌ **높은 CPC**: 경쟁 심화로 클릭 단가 상승
- ❌ **낮은 품질점수**: 랜딩페이지 관련성 낮음

- 배방 신축 아파트

- 배방 아파트 매매
