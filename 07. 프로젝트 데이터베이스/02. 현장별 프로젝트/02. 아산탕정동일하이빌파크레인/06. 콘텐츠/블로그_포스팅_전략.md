## 3. 콘텐츠 포맷 (Content Format)
### Type A: [분석형] "부동산 전문가의 시선"
*   **특징:** 객관적 데이터(지도, 시세표, 뉴스 기사) 활용, 논리적 설득
*   **예시 제목:**
    *   "아산 탕정, 지금 들어가도 될까? 향후 3년 공급 물량 분석"
    *   "동일하이빌 파크레인 vs 지웰시티, 전격 비교 분석 (가격/입지/평면)"

### Type B: [후기형] "내 집 마련 일기"
*   **특징:** 1인칭 시점의 스토리텔링, 감성적인 사진 위주
*   **예시 제목:**
    *   "주말에 남편이랑 탕정 동일하이빌 모델하우스 다녀왔어요 (84T 테라스 대박...)"
    *   "천안에서 탕정으로 이사 결심한 이유 (아이 교육 때문에)"

### Type C: [정보형] "동네 가이드"
*   **특징:** 맛집, 카페, 학원가 등 생활 정보 제공 (지역 커뮤니티 타겟)
*   **예시 제목:**
    *   "탕정 동일하이빌 입주민이 추천하는 단지 앞 산책 코스 BEST 3"
    *   "탕정역 근처 분위기 좋은 브런치 카페 모음"

## 4. OSMU (One Source Multi Use) 워크플로우

### 🔄 YouTube -> Blog 변환 프로세스
1.  **소스 영상:** 유튜브 '랜선 임장' 영상 (10분)
2.  **캡처:** 주요 스팟(정문, 학교, 공원) 고화질 캡처 10장
3.  **텍스트:** 영상 스크립트를 블로그 어투로 변환 (구어체 -> 문어체)
4.  **발행:** "영상으로 보는 탕정 동일하이빌 임장기" 포스팅 발행 (영상 링크 삽입)

### 🔄 Blog -> Instagram/Shorts 변환 프로세스
1.  **소스 글:** "84A타입 인테리어 꿀팁" 블로그 포스팅
2.  **요약:** 핵심 팁 3가지 추출 (카드뉴스 텍스트)
3.  **제작:**
    *   **인스타:** 카드뉴스 5장 제작 ("저장각! 30평대 넓게 쓰는 법")
    *   **쇼츠:** 블로그 사진 슬라이드쇼 + AI 보이스 내레이션 ("1분 만에 보는 84A 인테리어")
