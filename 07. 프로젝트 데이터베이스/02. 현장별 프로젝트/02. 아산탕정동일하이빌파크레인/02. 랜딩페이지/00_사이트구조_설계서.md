**Secondary KPI:** 상담 신청 양식 제출 2% 이상

## 📄 페이지별 상세 설계

### ✅ Phase 1: 핵심 페이지 (1-2일 구축)

#### 01. 메인 홈페이지 (Landing Page)
**목적:** 첫인상 + 핵심 메시지 전달 + CTA 유도
**예상 체류시간:** 30-60초

**섹션 구성:**
1. **Hero Section (히어로)**
   - 배경: 단지 조감도 또는 파크사이드 이미지
   - 헤드라인: "탕정의 완성을 누리는 파크사이드 라이프"
   - 서브 헤드라인: "전매제한 없음 | 신생아 우선공급 | 84T 테라스 12세대"
   - CTA: 📞 전화 상담 버튼 (클릭투콜)

2. **3대 특별혜택 (Quick Info)**
   - 💼 전매제한 없음
   - 👶 신생아 우선공급
   - 🏡 84T 테라스 희소

3. **핵심 강점 8가지 (Features Grid)**
   - 아이콘 + 짧은 설명
   - 삼성 직주근접, 파크사이드, GTX, 등

4. **평형별 안내 (Unit Types)**
   - 84A, 84B, 84T, 99A, 99B, 펜트하우스
   - 각 평형 클릭 → 상세 페이지 이동

5. **Floating CTA**
   - 모바일: 하단 고정 전화 버튼
   - 데스크톱: 우측 상담 신청 폼

---

#### 02. 3대 특별혜택 페이지
**목적:** 차별화 포인트 상세 설명

**섹션 구성:**
1. **혜택 1: 전매제한 없음 💼**
   - 설명: 계약금 10% 완납 후 즉시 전매 가능
   - 혜택: 투자 유연성 극대화
   - 비교: 규제 지역 vs 비규제 지역
   - CTA: "투자 상담 문의"

2. **혜택 2: 신생아 우선공급 👶**
   - 설명: 2세 미만 자녀 15-20% 최우선 배정
   - 조건: 입주자모집공고일 기준 2세 미만
   - 혜택: 당첨 확률 대폭 상승
   - CTA: "자격 확인 상담"

3. **혜택 3: 84T 테라스 희소성 🏡**
   - 설명: 전체 738세대 중 단 12세대
   - 특징: 아파트 + 단독주택 하이브리드
   - 장점: 층간소음 걱정 없는 마당
   - CTA: "테라스 평면 보기"

---

#### 03. 84T 테라스 특화 페이지
**목적:** 희소 상품 집중 홍보

**섹션 구성:**
1. **Hero: 테라스 실사 이미지**
   - 헤드라인: "단 12세대만의 특권, 84T 테라스 하우스"

2. **테라스 5대 장점**
   - 층간소음 걱정 없는 자유
   - 아이들 뛰어놀 수 있는 마당
   - 반려동물 키우기 최적
   - 홈가드닝 취미 가능
   - 매도 시 프리미엄 극대화

3. **평면도 & 면적 상세**
   - 전용면적: 84.9735㎡
   - 테라스 면적: XX㎡
   - 3D 평면도

4. **실제 입주자 후기 (예상)**
   - "아이가 마음껏 뛰어놀아요"
   - "단독주택 장점 + 아파트 편리함"

5. **CTA: 테라스 우선 상담**

---

#### 04. 전 평형 소개 페이지
**목적:** 평형별 상세 정보 제공

**섹션 구성:**
1. **평형 선택 탭**
   - 84A | 84B | 84T | 99A | 99B | 펜트하우스

2. **각 평형별 정보**
   - 전용면적 / 공급면적
   - 세대수 / 희소성
   - 평면도 이미지
   - 주요 특징 (4Bay, LDK, 등)
   - 추천 타겟 (신혼부부, 교체 수요, 등)
   - 예상 분양가 (범위)

3. **평형 비교표**
   - 면적, 세대수, 특징 한눈에 비교

---

### ✅ Phase 2: 보강 페이지 (3-4일 구축)

#### 05. 입지 환경 페이지
**목적:** 위치적 가치 증명

**섹션 구성:**
1. **지도 (Google Maps 임베드)**
   - 단지 위치 마커
   - 주요 시설 마커 (삼성, 역, 학교, 쇼핑몰)

2. **직주근접: 삼성 벨트**
   - 삼성디스플레이: 차량 10분
   - 삼성 나노시티: 차량 15분
   - 협력업체 밀집 지역

3. **광역 교통**
   - 천안아산역: 차량 10-15분
   - KTX/SRT 서울 30-40분
   - GTX-A 연결 예정

4. **생활 인프라**
   - 갤러리아백화점, 이마트 트레이더스
   - 모다아울렛, 지웰시티몰
   - 천안 불당 학원가

5. **교육 환경**
   - 인근 초·중·고 리스트
   - 어린이집, 유치원
   - 학원가 접근성

---

#### 06. 커뮤니티 시설 페이지
**목적:** 단지 내 편의시설 소개

**섹션 구성:**
1. **커뮤니티 시설 개요**
   - 헤드라인: "집 안에서 누리는 프리미엄 라이프"

2. **시설별 소개**
   - 피트니스센터 (24시간 운영)
   - 골프연습장 (스크린 골프)
   - 키즈카페 (실내 놀이터)
   - 독서실/스터디룸
   - 라운지/카페
   - 게스트하우스

3. **이미지 갤러리**
   - 각 시설 조감도 또는 레퍼런스 이미지

---

#### 07. 상담 신청 폼 페이지
**목적:** 리드 확보

**입력 항목:**
- 이름 (필수)
- 연락처 (필수)
- 관심 평형 (선택)
- 방문 희망일 (선택)
- 문의사항 (선택)

**CTA:** "무료 상담 신청하기"

**자동 응답:**
"신청이 완료되었습니다. 24시간 내 연락드리겠습니다."

---

#### 08. 분양 일정 안내 페이지
**목적:** 청약 프로세스 안내

**섹션 구성:**
1. **주요 일정 타임라인**
   - 모집공고일: 2025.09.19
   - 특별공급: 2025.09.29
   - 1순위: 2025.09.30
   - 2순위: 2025.10.01
   - 당첨자 발표: 2025.10.16
   - 계약: 2025.10.27-29

2. **청약 자격 안내**
   - 1순위 조건
   - 특별공급 조건
   - 신생아 우선공급 상세

3. **준비 서류**
   - 신분증, 청약통장
   - 가족관계증명서
   - 소득 증빙 (해당 시)

## 📊 전환 추적 설정

### Google Analytics 4 (GA4)
- **페이지뷰 추적:** 모든 페이지
- **이벤트 추적:**
  - 전화 버튼 클릭
  - 상담 신청 폼 제출
  - 평면도 다운로드
  - 외부 링크 클릭

### 네이버 애널리틱스
- **유입 경로 분석**
- **전환 깔때기 설정:**
  1. 메인 페이지 진입
  2. 상세 페이지 열람
  3. 전화 클릭 또는 폼 제출

- **별도 번호 할당:** 랜딩페이지 전용 번호
