# 📚 학습 노트

실전 프로젝트를 통해 배운 내용을 체계적으로 정리하는 공간

## 📝 작성 가이드

### 템플릿 사용
`/templates/학습노트_템플릿.md` 복사하여 시작

### 파일명 규칙
```
YYYY-MM-DD_주제_카테고리.md

예시:
2024-11-18_네이버광고_키워드선정_광고.md
2024-11-20_Figma_컴포넌트화_디자인.md
```

### YAML 속성 활용
```yaml
category: 광고/디자인/콘텐츠/분석/기술
difficulty: 초급/중급/고급
priority: 낮음/중/높음
status: 진행중/완료/보류
```

## 📊 분류 체계

### 카테고리별
- **광고**: 네이버, 카카오, 메타, 당근마켓
- **디자인**: Figma, 광고 소재, 영상
- **콘텐츠**: 유튜브, 블로그, SNS
- **분석**: GA4, 전환 추적, 데이터 해석
- **기술**: 웹사이트, 도구 활용, 자동화

### 난이도별
- **초급**: 기본 개념, 도구 사용법
- **중급**: 최적화, 전략 수립
- **고급**: 고급 기법, 시스템 구축

## 🔄 학습 사이클

```
실행 → 관찰 → 분석 → 문서화 → 적용
```

### 1. 실행
- 새로운 기법/도구 시도
- 캠페인 진행

### 2. 관찰
- 결과 데이터 수집
- 문제/성공 요인 발견

### 3. 분석
- 왜 그런 결과가 나왔는가?
- 다음에는 어떻게?

### 4. 문서화
- 학습노트에 기록
- 핵심 인사이트 추출

### 5. 적용
- 다음 프로젝트 반영
- 프레임워크 개선

## 💡 작성 팁

### 구체적으로
❌ "광고가 잘 안됐다"
✅ "키워드 'XX아파트 분양'의 CTR 0.5%로 목표(2%) 대비 저조. 광고문구에 가격 정보 없어서로 추정"

### 액션 중심
- 배운 점 → 다음 액션 명시
- 문제 → 해결 방안 제시

### 연결하기
- 관련 프로젝트 링크
- 참고 레퍼런스 연결
- 다른 학습노트 참조

## 📈 학습 내용 활용

### 주간 리뷰
- 이번 주 학습 노트 검토
- 공통 패턴 발견
- 다음 주 개선 계획

### 프로젝트 시작 시
- 관련 학습 노트 검색
- 과거 실수 방지
- 성공 패턴 재활용

### 프레임워크 개선
- 검증된 방법론 → 코어 프레임워크 반영
- 템플릿 업데이트
- 체크리스트 개선

## 📅 정기 작성

### 프로젝트 종료 시 (필수)
- 전체 프로젝트 회고
- 주요 학습 포인트
- 다음 적용 사항

### 주간 (권장)
- 이번 주 주요 활동
- 새로 배운 기법
- 발견한 문제

### 즉시 (권장)
- 중요한 인사이트 발견 시
- 실수/오류 발생 시
- 새로운 도구/기법 시도 시
