# 🎙️ 일레븐랩스(AI 보이스) 스크립트 변환 템플릿

> [!TIP]
> **목적**: AI가 텍스트를 읽을 때 숫자, 단위, 주소 등을 자연스러운 한국어 발음으로 읽도록 변환하는 규칙과 양식입니다.
> **사용법**: 원본 스크립트를 복사하여 아래 규칙에 따라 '발음형 텍스트'로 변환하세요.

---

## 1. 변환 규칙 가이드 (Calibration Rules)

### 🔢 숫자 및 단위 변환
| 원본 표기        | 변환 표기 (발음형) | 예시                                        |
| :--------------- | :----------------- | :------------------------------------------ |
| **면적 (㎡)**    | **제곱미터**       | 84㎡ → **팔십사 제곱미터**                  |
| **평형 (평)**    | **평형 / 평**      | 32평형 → **삼십이 평형**                    |
| **타입 (A/B/C)** | **알파벳 발음**    | 99A 타입 → **구구 에이 타입**               |
| **금액 (억/천)** | **한글 표기**      | 8억 4천 → **팔억 사천**                     |
| **동/호수**      | **개별 숫자+단위** | 204동 → **이백사 동** / 101호 → **백일 호** |

### 📍 주소 및 지번 변환
| 원본 표기         | 변환 표기 (발음형) | 예시                                       |
| :---------------- | :----------------- | :----------------------------------------- |
| **번지 (하이픈)** | **~의 ~번지**      | 갈산리 64-5 → **갈산리 육십사 의 오 번지** |
| **도로명 주소**   | **한글 표기**      | 탕정면 매곡리 → **탕정면 매곡리** (그대로) |

### 🔣 특수기호 및 문장 부호
| 원본 표기        | 변환 표기          | 설명                                   |
| :--------------- | :----------------- | :------------------------------------- |
| **· (가운뎃점)** | **, (쉼표)**       | 나열 시 쉼표로 변경하여 쉬어 읽기 유도 |
| **~ (물결)**     | **에서**           | 8억~9억 → **팔억에서 구억**            |
| **( ) 괄호**     | **삭제 or 풀어서** | (32평형) → **삼십이 평형**             |

| **( ) 괄호** | **삭제 or 풀어서** | (32평형) → **삼십이 평형** |

### 🚫 마크다운 및 서식 제거
* **일레븐랩스 입력용 텍스트는 순수 텍스트(Plain Text)여야 합니다.**
* 제목(#), 볼드체(**), 구분선(---), 인용구(>) 등 모든 마크다운 문법을 제거하세요.
* 문단 구분은 엔터(줄바꿈) 두 번으로 처리합니다.

## 2. 스크립트 변환 작업 공간

### [STEP 1] 원본 스크립트
*(여기에 원본 스크립트를 붙여넣으세요)*

```text
예: 전용 84㎡ 기준 시세는 8억 4천만 원~9억 원 선입니다.
```

### [STEP 2] 11Labs용 변환 스크립트
*(규칙을 적용하여 변환된 텍스트를 작성하세요)*

```text
예: 전용 팔십사 제곱미터 기준 시세는 팔억 사천만 원에서 구억 원 선입니다.
```

## 4. 톤 앤 매너 설정 (프롬프트 참고용)
* **Stability (안정성)**: 50% (너무 높으면 단조로움, 너무 낮으면 불안정)
* **Similarity (유사도)**: 75%
* **Style Exaggeration (스타일 과장)**: 0% (뉴스/브리핑 톤은 0 추천)
* **Speaker Boost**: ON
