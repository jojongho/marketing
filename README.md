# 🚀 올인원 마케팅 프레임워크

부동산 분양 현장을 위한 종합 마케팅 전략 프레임워크

## 📋 프로젝트 개요

### 목적
- **재사용 가능한 마케팅 프레임워크** 구축
- **실전 프로젝트를 통한 점진적 완성** (Learning by Doing)
- **개인 전문성 + 브랜드 차별화** 통합

### 핵심 가치
- 🎯 **효율성**: 재사용 가능한 템플릿과 프로세스
- 📊 **데이터 기반**: 성과 측정과 지속적 개선
- 🎨 **차별화**: 경쟁사와 다른 크리에이티브 접근
- 📚 **학습 중심**: 실행하면서 배우고 문서화

---

## 🗂️ 프로젝트 구조

```
📂 marketing/
├─ 📋 00_프레임워크_코어/          # 기본 템플릿, 체크리스트
├─ 🎨 01_브랜드_아이덴티티/         # 브랜드 전략, 차별화
├─ 🌐 02_랜딩페이지/               # Figma 디자인, 카피
├─ 🎯 03_광고_캠페인/              # 네이버, 카카오, 메타 등
├─ 📹 04_콘텐츠_전략/              # 유튜브, 블로그, SNS
├─ 🧑‍💼 05_퍼스널_브랜딩/            # 포트폴리오, 전문가 포지셔닝
├─ 📊 06_데이터_분석/              # GA4, 전환 추적
├─ 🗂️ 07_프로젝트_데이터베이스/     # 현장별 데이터, 자산 라이브러리
├─ 📚 08_학습_노트/                # 학습 내용 정리
├─ 🔖 09_레퍼런스/                 # 참고 자료, 벤치마킹
└─ 📄 templates/                  # Obsidian 템플릿
```

---

## 🎯 실행 우선순위

### Phase 1: 즉시 실행 (1-2주)
1. ✅ **브랜드 아이덴티티 구축**
   - 타겟 페르소나 정의
   - 차별화 전략 수립
   - 메시지 아키텍처

2. ✅ **랜딩페이지 제작**
   - Figma 디자인 시스템
   - 섹션별 카피 작성
   - Figma Site 퍼블리싱

3. ✅ **네이버 검색광고 준비**
   - 키워드 리서치
   - 광고 그룹 설계
   - 광고 문안 작성

### Phase 2: 단기 실행 (1-2개월)
4. **광고 캠페인 집행 & 최적화**
   - 네이버 검색광고 (우선)
   - 성과 데이터 분석
   - A/B 테스트 및 최적화

5. **콘텐츠 제작 시작**
   - 현장 영상 (드론, 주변 환경)
   - 블로그 포스팅
   - 소재 재활용

### Phase 3: 장기 전략 (3-6개월)
6. **유튜브 채널 성장**
   - 채널 전략 수립
   - 정기 콘텐츠 발행
   - 랜딩페이지 연동

7. **퍼스널 브랜딩 사이트**
   - WordPress CMS 구축
   - 현장 포트폴리오 연결
   - 전문가 포지셔닝

## 🛠️ 기술 스택

### 웹사이트
- **랜딩페이지**: Figma → Figma Site
- **퍼스널 브랜딩**: WordPress (CMS)
- **프로토타입**: Notion Site

### 마케팅 도구
- **광고**: 네이버 검색광고, 카카오 비즈니스, 메타, 당근마켓
- **분석**: GA4, 네이버 애널리틱스
- **상담**: 카카오 상담톡, CTI 전화 추적

### 제작 도구
- **디자인**: Figma
- **영상**: 드론 촬영, 편집 소프트웨어
- **문서**: Obsidian (지식 관리)

- 브랜드 인지도
- 포트폴리오 현장 수

## 📝 사용 가이드

### 새 프로젝트 시작하기
1. `templates/프로젝트_현장_템플릿.md` 복사
2. `07_프로젝트_데이터베이스/현장별_프로젝트/`에 새 폴더 생성
3. 프로젝트 정보 입력
4. 체크리스트 따라 진행

### 학습 내용 정리하기
1. `templates/학습노트_템플릿.md` 사용
2. `08_학습_노트/`에 저장
3. 태그로 분류 관리

### 레퍼런스 추가하기
1. `templates/레퍼런스_템플릿.md` 사용
2. `09_레퍼런스/` 카테고리별 폴더에 저장

### 광고 캠페인 실행하기
1. `templates/광고_캠페인_템플릿.md` 사용
2. `03_광고_캠페인/` 플랫폼별 폴더에 저장
3. 일일 모니터링 및 최적화

## 🎨 차별화 전략

### 일반 마케팅 vs 우리의 접근
| 일반적 접근 | 차별화 접근 |
|------------|-------------|
| 📄 조감도 재활용 | 🎥 드론 항공촬영 (실제 환경) |
| 📝 표준 카피 반복 | ✍️ 맞춤형 스토리텔링 |
| 🖼️ 정적 이미지 | 🎬 영상 콘텐츠 |
| 📢 일방향 광고 | 🤝 콘텐츠 + 교육 + 신뢰 |

### 핵심 차별화 요소
- **비주얼**: 드론 촬영, 360도 VR, 생활환경 영상
- **콘텐츠**: 유튜브 채널, 블로그, 전문가 포지셔닝
- **데이터**: 체계적인 성과 추적 및 최적화
- **시스템**: 재사용 가능한 프레임워크

## 💡 프로젝트 원칙

1. **실행하면서 배운다** - 완벽을 기다리지 않고 시작
2. **데이터가 답이다** - 추측보다 측정과 분석
3. **재사용을 고려한다** - 모든 자산은 재사용 가능하게
4. **차별화가 핵심이다** - 남들과 같은 것은 의미 없음
5. **문서화는 필수다** - 배운 것은 반드시 기록
