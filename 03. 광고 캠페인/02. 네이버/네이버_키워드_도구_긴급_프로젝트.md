## 🛠️ 개선된 도구 사용법

### 새로운 키워드 조합기 v2

**파일 위치**: `03. 광고 캠페인/02. 네이버/03. 도구 개발/네이버_키워드_조합기_v2.xlsx`

**시트 구성**:
1. **DB_부동산**: 부동산 키워드 데이터베이스
2. **DB_생활관심**: 에펠탑용 생활 키워드
3. **DB_현장특화**: 현장명, 브랜드, 시설
4. **조합_실행**: 메인 작업 시트 (자동 조합 수식)
5. **네이버_업로드용**: CSV 추출용
6. **📖_사용가이드**: 상세 매뉴얼

### 빠른 시작 3단계

```
1. DB 시트에 키워드 입력
   → A, B, C 열에 기본 키워드 정리

2. 조합_실행 시트에서 복사/붙여넣기
   → E, F 열에서 자동 조합 확인

3. 네이버_업로드용에 붙여넣기 + CSV 저장
   → UTF-8 BOM 인코딩 필수!
```

```
배방 아파트 분양
배방역 아파트
배방읍 아파트
...
```

### 🗼 에펠탑 키워드 (500개 목표)

**목적**: 반복 노출, 브랜드 각인

**카테고리별 목표 수량**:
- 생활/음식: 150개 (배방 카페, 배방 맛집, 배방 치킨...)

- 기타 관심사: 100개 (배방 헬스장, 배방 영어학원...)
