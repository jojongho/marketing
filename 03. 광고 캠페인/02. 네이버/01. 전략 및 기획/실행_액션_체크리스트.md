### 🎯 3시간 내 완료

#### CSV 생성

- [ ] **1. 조합 결과 복사**
  - 조합_실행 시트의 F열(3단조합) 복사
  - 값만 복사 (수식 제외)

- [ ] **2. 네이버_업로드용 시트에 붙여넣기**
  - B열(키워드)에 붙여넣기
  - A열(광고그룹ID): 비워두기
  - C열(입찰가):
    - 브랜드/고단가: 150~200 입력
    - 에펠탑: 70 입력
  - D열(매칭옵션): "확대" 입력
  - E열(사용여부): "Y" 입력

- [ ] **3. 중복 제거**
  - 데이터 → 중복 항목 제거
  - 키워드 열 기준

- [ ] **4. 불필요 키워드 삭제**
  - "배방읍 회룡리 아파트" 같은 존재하지 않는 조합
  - 너무 일반적인 키워드 (예: "아파트", "분양")

- [ ] **5. CSV 저장**
  ```
  파일명: 배방우방_키워드_[N]개_20241128.csv
  인코딩: UTF-8 (BOM) ← 필수!

  Excel에서:
  다른 이름으로 저장 → CSV UTF-8 (쉼표로 분리)
  ```
