# 키워드 조합기 및 자동화 도구

이 폴더는 네이버 검색광고 효율화를 위한 도구를 개발하고 관리하는 공간입니다.

## 개발 예정 도구
### 1. 키워드 조합기 (Keyword Combiner)
- **기능**: 시드 키워드와 확장 키워드를 조합하여 대량의 키워드 리스트 생성
- **형식**: Python 스크립트 또는 엑셀 매크로
- **입력**: 메인 키워드, 서브 키워드, 지역명 등
- **출력**: 조합된 키워드 리스트 (CSV/XLSX)

### 2. 네이버 광고 API 연동 (선택 사항)
- 광고 성과 자동 수집
- 입찰가 자동 조정
