# 엑셀 함수로 만드는 초간단 키워드 조합기 (MVP)

이 가이드는 복잡한 프로그래밍 없이 **엑셀 기본 함수**만을 사용하여 키워드를 조합하는 방법을 설명합니다.

## 1. 기본 구조 (1:1 조합)
가장 단순한 형태입니다. A열과 B열을 순서대로 합칩니다.

### 시트 구성
- **A열**: 메인 키워드 (예: 강남, 홍대, 이태원)
- **B열**: 서브 키워드 (예: 맛집, 카페, 술집)
- **C열**: 조합 결과

### 함수 (C2 셀)
```excel
=A2 & " " & B2
```
*설명: A2셀과 B2셀 사이에 공백을 넣어 합칩니다.*

---

## 2. 고급 구조 (N:M 모든 경우의 수 조합)
메인 키워드 10개와 서브 키워드 10개가 있을 때, 100개의 조합을 자동으로 만드는 방법입니다.

### 시트 구성
- **A열**: 메인 키워드 리스트 (데이터 소스)
- **B열**: 서브 키워드 리스트 (데이터 소스)
- **D열**: 조합 결과 (출력)

### 함수 (D2 셀에 입력 후 아래로 드래그)
이 함수는 A열과 B열의 모든 가능한 조합을 자동으로 생성합니다.

```excel
=INDEX(A:A, INT((ROW()-2)/COUNTA(B:B))+2) & " " & INDEX(B:B, MOD(ROW()-2, COUNTA(B:B))+2)
```

### 함수 설명
1. **`COUNTA(B:B)`**: B열에 있는 서브 키워드의 개수를 셉니다.
2. **`ROW()-2`**: 현재 행 번호를 기준으로 0부터 시작하는 인덱스를 만듭니다. (2행부터 시작하므로 -2)
3. **`INT(...)`**: 메인 키워드(A열)를 순환시키는 인덱스를 계산합니다.
4. **`MOD(...)`**: 서브 키워드(B열)를 반복시키는 인덱스를 계산합니다.
5. **`INDEX(...)`**: 계산된 인덱스에 해당하는 실제 단어를 가져옵니다.

## 3. 글자수 확인 (네이버 광고 제약)
네이버 검색광고는 키워드 글자수 제한(최대 25자)이 있습니다.

### 함수 (E2 셀)
```excel
=LEN(D2)
```
*팁: 조건부 서식을 사용하여 25자가 넘으면 빨간색으로 표시하면 편리합니다.*

## 4. 바로 시작하기
1. 빈 엑셀 파일을 엽니다.
2. A열, B열에 단어들을 채워 넣습니다.
3. D2 셀에 위 '고급 구조' 함수를 복사해 넣습니다.
4. D2 셀의 채우기 핸들을 잡고 원하는 만큼 아래로 쭉 드래그합니다.

---

## 5. [심화] 3개 이상 그룹 조합하기 (계단식 접근법)
키워드 그룹이 3개, 4개로 늘어나면 한 번에 수식을 짜기 매우 복잡해집니다.
이때는 **중간 결과**를 만드는 '계단식' 방법을 사용하면 누구나 쉽게 만들 수 있습니다.

### 예시: 지역 + 업종 + 세부키워드 (3단 조합)
- **A열**: 지역 (강남, 홍대...)
- **B열**: 업종 (맛집, 카페...)
- **C열**: 세부 (추천, 데이트...)

#### 1단계: A + B 조합하기 (E열)
먼저 앞서 배운 '고급 구조' 수식을 사용하여 A와 B를 조합해 E열에 만듭니다.
- **E열 수식**: `=INDEX(A:A, ...)` (위 2번 항목 참조)
- 결과: "강남 맛집", "강남 카페", "홍대 맛집"...

#### 2단계: (A+B) + C 조합하기 (F열)
이제 **만들어진 E열**을 새로운 '메인 키워드'로 생각하고 C열과 합칩니다.
- **F열 수식**:
```excel
=INDEX(E:E, INT((ROW()-2)/COUNTA(C:C))+2) & " " & INDEX(C:C, MOD(ROW()-2, COUNTA(C:C))+2)
```
*핵심: 수식에서 참조하는 열만 A, B에서 → E, C로 바꿔주면 됩니다.*

### 4단, 5단 조합도 동일합니다
- 1차 조합: A + B = **AB**
- 2차 조합: **AB** + C = **ABC**
- 3차 조합: **ABC** + D = **ABCD**

이 방식을 사용하면 엑셀 함수만으로도 무한대의 조합을 오류 없이 만들 수 있습니다.

## 6. 옵션: 공백 제거하기
키워드 사이에 공백을 넣고 싶지 않다면(예: 강남맛집), 수식 가운데의 `& " " &` 부분을 `&` 로만 바꾸면 됩니다.
- 공백 있음: `=A2 & " " & B2`
- 공백 없음: `=A2 & B2`
