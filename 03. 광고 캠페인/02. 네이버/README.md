## 🎯 전략 개요

### 목표
- **Primary**: 전화 문의 유도
- **Secondary**: 상담 신청, 관심고객 등록

### 타겟 지역
- 천안시 (동남구, 서북구)
- 아산시
- 주변 수도권 (서울 출퇴근 가능 지역)

### 예산 가이드
- **테스트 단계**: 일 3-5만원 (주 21-35만원)
- **안정화 단계**: 일 5-10만원
- **스케일업**: 성과 기반 확대

## 🔑 키워드 전략

### 카테고리별 키워드

#### 1. 브랜드 키워드 (필수)
```
[현장명]
[현장명] 분양
[현장명] 모델하우스
[브랜드명] [지역명]
```
**특징**: 낮은 CPC, 높은 전환율, 경쟁사 유입 방지

#### 2. 지역 + 일반 키워드 (핵심)
```
천안 아파트 분양
아산 신축 아파트
천안역 오피스텔
[역명] 아파트
```
**특징**: 검색량 많음, 경쟁 높음, 핵심 타겟

#### 3. 특성 키워드
```
천안 초품아 아파트
아산 역세권 분양
천안 학군 좋은 아파트
천안 공원 근처 아파트
```
**특징**: 현장 강점 부각, 질 높은 트래픽

#### 4. 롱테일 키워드
```
천안 [동네명] 분양 아파트
아산 [특정지역] 신축
천안 [학교명] 학군 아파트
```
**특징**: 낮은 경쟁, 구매 의도 높음

## 📊 광고 그룹 구조 예시

```
캠페인: [현장명] 분양 마케팅
│
├─ 그룹 1: 브랜드 키워드
│  ├─ [현장명]
│  ├─ [현장명] 분양
│  └─ [현장명] 모델하우스
│  입찰: 높음 (브랜드 방어)
│
├─ 그룹 2: 지역 + 분양
│  ├─ 천안 아파트 분양
│  ├─ 아산 신축 아파트
│  └─ 천안 분양 정보
│  입찰: 중상
│
├─ 그룹 3: 역세권
│  ├─ [역명] 아파트
│  ├─ 천안 역세권 분양
│  └─ 아산 역세권 오피스텔
│  입찰: 높음 (핵심 강점)
│
├─ 그룹 4: 특성 (학군/공원 등)
│  ├─ 천안 초품아
│  ├─ 천안 학군 아파트
│  └─ 아산 공원 아파트
│  입찰: 중
│
└─ 그룹 5: 롱테일
   ├─ 천안 [동네] 분양
   └─ 기타 롱테일 키워드
   입찰: 낮음-중
```

## ✍️ 광고 문안 작성 원칙

### 제목 (30자 이내)
**구조**: [지역/역명] + [핵심가치] + [차별화]

**예시**:
```
✅ "천안역 도보 3분, XX아파트 분양"
✅ "초품아 프리미엄! XX 모델하우스"
✅ "천안 핵심입지 XX 특별분양"

❌ "좋은 아파트 분양합니다" (너무 일반적)
❌ "XX아파트 123동 456호" (무의미한 세부사항)
```

### 설명 (45자 이내)
**구조**: [구체적 혜택] + [CTA]

**예시**:
```
✅ "역세권+명문학군 완벽입지. 지금 모델하우스 방문하세요"
✅ "84㎡~105㎡ 다양한 평형. 특별 분양 상담 예약 중"
✅ "공원+학교+쇼핑몰 도보권. 전화문의 시 사은품 증정"

❌ "좋은 위치에 있습니다. 연락주세요" (구체성 없음)
```

## 📈 성과 측정

### 주요 지표
| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| CTR | 2% 이상 | 네이버 광고 관리 |
| 전화 문의 | 주 10건+ | CTI 추적 |
| CPA | 5만원 이하 | 비용/전환수 |
| 전환율 | 3% 이상 | 클릭/전환 |

### 일일 체크리스트
- [ ] 노출수/클릭수 확인
- [ ] 전화 문의 수 기록
- [ ] CPC 변동 모니터링
- [ ] 광고 게재 상태 확인
- [ ] 예산 소진율 체크

### 주간 최적화
- [ ] 성과 낮은 키워드 일시정지
- [ ] 성과 높은 키워드 입찰 상향
- [ ] 광고문안 A/B 테스트
- [ ] 새 키워드 추가
- [ ] 랜딩페이지 개선

## 🎨 크리에이티브 최적화

### 광고 확장 소재 활용
- **전화번호**: 필수 (클릭 시 바로 통화)
- **부가 정보**: 위치, 영업시간, 주차 가능
- **사이트링크**: 모델하우스, 평면도, 위치, 상담예약

### 이미지 광고 (파워링크 이미지)
- 조감도 또는 드론 촬영 이미지
- 깔끔한 텍스트 오버레이
- 주요 강점 3개 강조

## 💡 실전 팁

### 시작 단계 (1-2주)
1. 소규모 예산으로 시작 (일 3만원)
2. 5-7개 광고 그룹 생성
3. 각 그룹당 2-3개 광고문안 테스트
4. 매일 성과 모니터링

### 최적화 단계 (2-4주)
1. 저성과 키워드 정리
2. 고성과 키워드 예산 증액
3. 광고문안 개선
4. 입찰가 미세조정

### 스케일업 단계 (4주+)
1. 예산 점진적 확대
2. 신규 키워드 발굴
3. 타겟 지역 확대
4. 리마케팅 추가

## ⚠️ 주의사항

### 피해야 할 것
- ❌ 처음부터 고예산 집행
- ❌ 너무 많은 키워드 (관리 불가)
- ❌ 일반적인 광고문안
- ❌ 데이터 없는 의사결정

### 반드시 해야 할 것
- ✅ 전화 추적 설정

- ✅ 매일 성과 확인
- ✅ 주간 리포트 작성
- ✅ 경쟁사 모니터링
