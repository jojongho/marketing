## 💡 핵심 원칙 (리마인더)

1. **실행하면서 배운다**
   - 완벽을 기다리지 않고 시작
   - 작은 것부터 빠르게 테스트

2. **데이터가 답이다**
   - 추측보다 측정과 분석
   - 매일 성과 확인

3. **재사용을 고려한다**
   - 모든 자산은 재사용 가능하게
   - 템플릿과 프로세스 지속 개선

4. **차별화가 핵심이다**
   - 남들과 같은 것은 의미 없음
   - 창의적이고 독특한 접근

5. **문서화는 필수다**
   - 배운 것은 반드시 기록
   - 학습 노트로 지속 성장

## 🎨 차별화 전략 (재확인)

### 일반 vs 우리
| 일반 마케팅 | 우리의 차별화 |
|------------|--------------|
| 조감도 재활용 | 드론 항공촬영 |
| 표준 카피 | 맞춤 스토리텔링 |
| 정적 이미지 | 영상 콘텐츠 |
| 일방향 광고 | 콘텐츠 + 교육 |

### 실행 포인트
- **비주얼**: 실제 환경을 보여주는 영상
- **콘텐츠**: 유튜브로 신뢰 구축
- **데이터**: 체계적 추적과 최적화
- **시스템**: 프레임워크로 효율화
