# ⚡ Quick Start Guide

프로젝트를 바로 시작하기 위한 단계별 가이드

## 🎯 지금 바로 시작하기

### 1단계: 첫 프로젝트 생성 (5분)

```bash
# 1. 프로젝트 폴더 생성
07_프로젝트_데이터베이스/현장별_프로젝트/[현장명]_2024-11/

# 2. 템플릿 복사
templates/프로젝트_현장_템플릿.md → 위 폴더에 복사

# 3. 기본 정보 입력
- 현장명, 지역, 브랜드명
- 예상 기간, 예산
- 핵심 강점 3가지
```

### 2단계: 타겟 정의 (10분)

**질문에 답하세요:**
- 누구에게 팔 것인가? (30-40대 실거주? 투자자?)
- 왜 이 현장인가? (역세권? 학군? 투자가치?)
- 경쟁 현장 대비 우리 장점?

**작성 위치**: `01_브랜드_아이덴티티/타겟_페르소나_천안아산.md`

### 3단계: 네이버 키워드 리서치 (20분)

1. [네이버 키워드 도구](https://searchad.naver.com/) 접속
2. 핵심 키워드 10개 검색
   - [지역명] 아파트 분양
   - [지역명] [특성] 아파트
   - [역명] 아파트
3. 검색량, 경쟁도 기록

**작성 위치**: `03_광고_캠페인/네이버_검색광고/[현장명]_키워드_리서치.md`

### 4단계: 랜딩페이지 기획 (30분)

1. `02_랜딩페이지/섹션별_카피_템플릿/분양_랜딩페이지_프롬프트_가이드.md` 열기
2. 변수 값 치환
   - {현장명}, {지역명}, {역명} 등
3. 각 섹션 카피 작성

**Figma 작업 시작**

### 5단계: 광고 준비 (20분)

1. `templates/광고_캠페인_템플릿.md` 복사
2. 키워드 리서치 결과 반영
3. 광고 문안 3개 작성
4. 예산 설정 (일 3-5만원 추천)

**네이버 광고 관리 시스템 설정**

## 🎨 Figma 빠른 시작

### 디자인 시스템 설정
1. 컬러 팔레트 (5-7개)
   - Primary: 브랜드 메인 컬러
   - Secondary: 강조 컬러
   - Neutral: 배경, 텍스트

2. 타이포그래피
   - Heading: 40-56px (Bold)
   - Body: 16-18px (Regular)
   - Caption: 14px (Light)

3. 컴포넌트 생성
   - Button (Primary, Secondary)
   - Input Form
   - Card
   - Section Container

### 랜딩페이지 섹션
1. **Hero**: 헤드라인 + 서브카피 + CTA + 비주얼
2. **About**: 현장 소개 + 핵심 강점
3. **Features**: 8개 강점 그리드
4. **CTA**: 전화/상담 신청
5. **Footer**: 필수 정보

## 📢 네이버 광고 빠른 시작

### 최소 구성 (3개 그룹)
```
그룹 1: 브랜드 키워드 (2-3개)
→ [현장명], [현장명] 분양

그룹 2: 지역 키워드 (5-7개)
→ 천안 아파트 분양, 아산 신축 아파트

그룹 3: 특성 키워드 (3-5개)
→ 천안 역세권, 아산 초품아
```

### 광고 문안 템플릿
```
제목: [지역/역명] + [핵심가치] (30자)
설명: [구체적 혜택] + [CTA] (45자)

예시:
제목: "천안역 3분거리 XX아파트 특별분양"
설명: "84㎡~105㎡ 프리미엄 평형. 지금 상담예약 시 사은품"
```

### 일일 루틴 (5분)
1. 네이버 광고 관리 접속
2. 노출/클릭/전환 확인
3. 전화 문의 수 기록
4. 이상 징후 체크

## 💡 자주 묻는 질문

### Q1. 예산은 얼마나 필요한가요?
**A**: 테스트 단계는 일 3-5만원 추천. 2주 후 성과 보고 조정.

### Q2. 랜딩페이지 제작 시간은?
**A**: Figma 익숙하면 1-2일. 처음이면 3-5일 예상.

### Q3. 광고는 언제 시작하나요?
**A**: 랜딩페이지 완성 후 즉시. 동시 제작 가능.

### Q4. 경쟁사는 몇 개나 분석하나요?
**A**: 2-3개만 깊게. 너무 많으면 혼란.

### Q5. 성과는 언제부터 나오나요?
**A**: 전화 문의는 당일부터 가능. 최적화는 2-4주 필요.
